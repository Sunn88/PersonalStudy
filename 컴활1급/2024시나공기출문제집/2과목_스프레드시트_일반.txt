099. 엑셀의 주요 화면 구성
- 리본 메뉴
    - 탭, 그룹, 명령으로 구성
    - 탭 종류 : 홈, 삽입, 페이지 레이아웃, 수식, 데이터, 검토, 보기, 개발 도구 등
    - Alt + F10 : 리본 메뉴에 바로 가기 키가 표시됨
- 이름 상자
    - 현재 작업중인 셀의 이름이나 주소 표시 부분
    - 차트 항목이나 그리기 개체 선택시 개체 이름이 표시됨
- 상태 표시줄
    - 현재 작업 상태나 선택한 명령에 대한 기본적인 정보 표시 부분
    - 상태 표시줄 사용자 지정 : 평균, 개수(데이터가 입력된 셀), 숫자 셀 수, 최소값, 최대값, 합계를 선택해 자동 계산 가능
    - 매크로 기록 : 매크로 기록 중에는 '기록 중지' 아이콘으로 변경됨
    - 보기 바로 가기 : 워크시트의 보기 상태를 변경할 수 있음
        - 기본
        - 페이지 레이아웃- 페이지 나누기 미리 보기
    - 확대/축소 슬라이더 : 화면을 10~400%로 확대/축소할 수 있는 컨트롤

100. 워크시트 편집
- 워크시트 선택
    - 연속적인 여러 개의 시트 선택 : shift누른 채 마지막 시트 선택
    - 비연속적인 여러 개의 시트 : ctrl을 누른 채 원하는 시트 차례로 클릭
    - 여러 개의 시트를 선택하면 제목 표시줄에 '그룹'이라고 표시됨
    - 여러 개의 시트를 선택하고 데이터를 입력하면 선택한 모든 시트에 동일한 데이터가 입력됨
    - 그룹 상태에서는 그래픽 개체를 삽입하거나 데이터 관리 작업을 수행할 수 없음
    - 그룹 상태를 해제하려면 시트 탭의 바로 가기 메뉴에서 '시트 그룹 해제'를 선택하거나 임의의 시트를 클릭함
- 워크시트 이름 변경
    - 바꿀 이름을 더블클릭한 후 원하는 이름 입력 후 엔터
    - 공백을 포함해 최대 31자까지 지정 -> * / : ? [] 등의 문자는 사용 불가
- 워크시트 삽입
    - 메모리 한도까지 워크시트 추가 가능
    - 삽입된 시트는 활성 시트의 왼쪽에 삽입됨
    - 시트 삽입 바로 가기 : shift + F11
- 워크시트 이동 : 이동할 시트 클릭 후 원하는 위치까지 드래그
- 워크시트 복사
    - 복사할 시트 선택 후 ctrl을 누른 채 원하는 위치까지 드래그
    - 복사할 때마다 시트 이름 뒤에 () 삽입됨
- 워크시트 삭제
    - 삭제할 시트 선택 후 홈->셀->삭제->시트 삭제
    - 시트 탭의 바로 가기 메뉴에서 '삭제'선택
    - 되살리기 불가
    - 여러개 동시 삭제 가능

101. 데이터 입력
- 셀 안에서 줄을 바꾸려면 alt+enter
- 여러 셀에 동일 내용 입력하려면 해당 셀 범위로 지정 후 데이터 입력하고 ctrl+enter
- 범위 지정 후 enter 누르면 지정 범위 안에서만 셀 포인터 이동
- 셀 선택후 alt+아래화살표 : 같은 열에 입력된 문자열 목록 표시
- 셀 내용 자동 완성
    - 데이터 입력 중 처음 몇 자가 같은 열에 이미 입력된 내용과 동일하면 자동으로 나머지 내용이 채워짐
    - 문자에만 적용됨, 숫자/날짜/시간 형식의 데이터에는 적용안됨
    - 자동 완성 기능 사용 : 파일->옵션->고급->편집 옵션->'셀 내용을 자동 완성'

102. 데이터 형식
- 문자 데이터
    - 기본적으로 셀 왼쪽 정렬
    - 숫자 데이터 앞에 문자 접두어(') 입력시 문자 데이터로 인식됨
- 수치 데이터
    - 기본적으로 셀 오른쪽 정렬
    - 분수는 0 입력후 한칸 띄운 다음 입력 ex) 0 1/2
    - 셀 너비보다 긴 경우 지수 형식으로 표시됨
- 날짜 데이터
    - 하이픈(-)이나 슬래시(/)를 이용해 연, 월, 일 구분
    - 오늘 날짜 입력 : ctrl+;
- 시간 데이터
    - 콜론(;)을 이용해 시, 분, 초 구분
    - 오늘 시간 입력 : ctrl+shift+;
- 노트
    - 셀에 입력된 데이터를 지워도 노트는 삭제되지 않음
    - 시트에 삽입된 노트를 시트에 표시된 대로 인쇄하거나 시트 끝에 모아서 인쇄할 수 있음
    - 노트의 위치를 자유롭게 이동하거나 노트가 항상 표시되도록 지정할 수 있음
    - 셀에 입력된 데이터를 정렬하면 노트도 함께 이동되지만, 피벗 테이블에 삽입된 노트는 이동하지 않음
- 윗주
    - 셀에 입력된 데이터의 위쪽에 추가하는 주석문 -> 문자 데이터만 삽입 가능
    - 윗주가 삽입된 셀의 데이터 삭제시 윗주도 함게 삭제됨
    - 셀에 수치 데이터, 날짜/시간 데이터가 입력되어 있거나 아무것도 없으면 윗주 삽입 불가
    - 윗주 서식은 윗주 전체에 대해서만 적용하거나 변경 가능

103. 채우기 핸들을 이용한 연속 데이터 입력
- 숫자 데이터
    - 한 셀 : 드래그할 경우 동일한 데이터가 복사됨, ctrl누르고 드래그하면 값이 1씩 증가
    - 두 셀 : 첫 셀과 두 번째 셀의 차이만큼 증감함
- 사용자 지정 목록
    - 사용자 지정 목록에 등록된 문자 데이터 중 하나를 입력하고 드래그하면 등록된 순서대로 반복되어 입력됨
    - 파일->옵션 클릭 후 'excel 옵션'의 고급 탭에서 일반 항목의 '사용자 지정 목록 편집'에서 추가/삭제
- 문자 데이터
    - 한 셀 : 드래그하면 동일한 데이터 입력됨
    - 두 셀 : 드래그하면 두 개의 문자가 반복 입력됨
- 혼합 데이터(문자 + 숫자)
    - 한 셀 : 가장 오른쪽에 있는 숫자는 1씩 증가, 나머지는 그대로 입력
    - 두 셀 : 숫자 데이터는 차이만큼 증/감, 문자는 그대로 입력
- 날짜 데이터
    - 한 셀 : 1일 단위로 증가
    - 두 셀 : 두 셀의 차이만큼 연, 월, 일 단위로 증가

104. 찾기
- 워크시트에 입력되어 있는 데이터 중 특정 내용을 찾는 기능 -> 숫지, 특수문자, 한자 등도 가능
- 워크시트 전체를 대상으로 찾거나 범위 지정 가능
- 여러 개의 워크시트를 선택하고 찾기 실행시 하나의 워크시트에 있는 것처럼 연속 찾기 실행
- '찾기 및 바꾸기' 대화상자
    - 범위 : 찾을 범위 -> 시트나 통합 문서 지정
    - 검색 : 찾을 방향 -> 행이나 열로 지정
    - 찾는 위치 : 찾을 정보가 들어 있는 워크시트의 요소 -> 수식, 값, 슬라이드 노트, 메모 지정
    - 대/소문자 구분
    - 전체 셀 내용 일치 : 찾을 내용과 완전히 일치하는 셀만을 찾음
- 데이터를 역순 검색하려면 shift를 누른 상태에서 다음 찾기 클릭
- *, ? 등의 만능문자 사용 가능 -> 만능문자 자체를 검색하려면 기호 앞에 ~를 입력함

105. 셀 포인터 이동
- shift+tab, tab : 좌우 이동
- shift+enter, enter : 상하 이동
- home : 해당 행 A열로 이동
- ctrl+home : a1셀로 이동
- ctrl+end : 데이터 범위의 맨 오른쪽 아래 셀로 이동
- ctrl+화살표 : 데이터 범위의 상하좌우 끝으로 이동
- pgup, pgdn : 한 화면 위아래로 이동
- alt+pgup, alt+pgdn : 한 화면 좌우이동
- ctrl+pgup, ctrl+pgdn : 현재 시트의 앞뒤시트로 이동
- f5 : 이동하고자 하는 셀 주소 직접 입력

106. 파일->옵션의 '고급'탭
- 편집 옵션
    - enter 키를 누른 후 다음 셀로 이동 : 이동 방향 지정
    - 소수점 자동 삽입 : 입력한 숫자 데이터의 소수점 위치 설정
    - 셀에서 직접 편입 허용 : 셀을 더블클릭해 데이터의 수정이 가능하도록 설정
    - 셀 내용 자동 완성
    - 마우스로 화면 확대/축소
- 이 워크시트의 표시 옵션
    - 행 및 열 머리글 표시 : 표시 여부 지정
    - 페이지 나누기 표시 : 자동으로 표시되는 페이지 나누기 선 표시 여부 지정
    - 0 값이 있는 셀에 0 표시 : 표시 여부 지정
    - 눈금선 표시 : 표시 여부 지정

107. 셀 삽입/삭제/병합
- 셀 삽입
    - 기존 셀을 오른쪽이나 아래쪽으로 밀어내고 지정한 범위만큼 새로운 셀 삽입
    - 삽입할 셀 범위 지정한 다음 ctrl++ 누른 후 '삽입' 대화상자가 나타나면, 기존에 있던 데이터의 이동방향 선택하고 '확인'클릭
- 셀 삭제
    - 범위로 지정한 셀 삭제, 아래쪽이나 오른쪽에 있는 셀 삭제한 자리로 이동하는 기능
    - 삽입할 셀 범위 지정한 다음 ctrl+- 누른 후 '삭제' 대화상자가 나타나면, 기존에 있던 데이터의 이동방향 선택하고 '확인'클릭
- 셀 병합
    - 여러 개의 셀을 하나로 합치는 기능
    - 홈->맞춤->병합하고 가운데 맞춤
    - 데이터가 입력되어 있는 여러 개의 셀을 병합할 경우 가장 위쪽 또는 왼쪽의 데이터만 남고 나머지 셀의 데이터는 모두 지워짐

108. 통합 문서 공유
- 네트워크로 연결된 환경에서 하나의 통합 문서를 여러 사람이 공동으로 작업할 수 있게 하는 기능
- 통합 문서를 공유한 후 불가능한 작업 : 셀 병함, 조건부 서식, 차트, 시나리오, 부분합, 데이터 표, 피벗 테이블 보고서 등의 추가 및 변경
- 공유된 통합 문서는 제목 표시줄에 '공유됨'이라고 표시됨
- 공유 통합 문서의 변경 내용을 추적해 변경 내용만을 모아 새로운 시트에 작성할 수 있음
- 공유된 통합 문서는 여러 사용자가 동시에 변경 및 병합할 수 있음
- 공유 통합 문서를 보호하기 위해 암호를 설정할 수 있음
- 암호로 보호된 공유 통합 문서의 보호를 해제하려면 먼저 통합 문서의 공유를 해제해야 함
- 공유 통합 문서 파일을 다른 위치에 복사해도 공유 설정 값은 유지됨
- 공유 통합 문서가 저장된 네트워크 위치를 액세스하는 모든 사용자는 공유 통합 문서를 액세스할 수 있음
- 공유 통합 문서의 변경 내용을 일정 기간 동안 보관할 수 있으며, 그 여부를 지정할 수 있음
- 변경 내용을 저장하면 공유 통합 문서의 복사본이 만들어져 변경한 내용들을 병합할 수 있음
- 공유 통합 문서를 사용하는 여러 사용자들의 변경 내용이 충돌할 경우 저장할 내용을 선택하거나, 자신이 변경한 내용이 무조건 저장되도록 지정할 수 있음

109. 시트 보호
- 특정 워크시트에 입력된 데이터나 차트 등을 변경할 수 없도록 보호하는 것
- 보호된 시트에서는 기본적으로 셀을 선택하는 것만 가능
- 검토->보호->시트 보호/해제
- 통합 문서 중 특정 시트만을 보호하는 것으로, 나머지 시트는 변경 가능
- 모든 요소를 모든 사용자가 액세스하지 못하도록 보호 가능
- 지정한 범위에 대해 개별적으로 사용자의 수정을 허용할 수 있음
- 셀/행/열의 서식, 하이퍼링크 삽입, 자동 필터 사용, 피벗 테이블 및 피벗 차트 사용, 개체 편집, 시나리오 편집 등 특정 항목을 제외하고 시트 보호 지정 가능
- 잠금이 해제된 셀은 보호되지 않음

110. 통합 문서 보호
- 통합 문서의 시트 삽입/삭제/이동/숨기기/이름 바꾸기 등을 할 수 없게 보호
- 검토->보호->통합 문서 보호
- '시트 보호;가 설정되지 않은 경우 워크시트에 데이터를 입력/수정/삭제하거나 피벗 테이블 보고서, 부분합과 같은 데이터 분석 작업 가능
- 암호 지정 가능

111. 사용자 지정 서식
- 조건이 없을 때는 양수-음수-0-텍스트 순으로 표시 형식 지정됨
- 조건이 있을 때는 조건 지정 순으로 표시 형식 나타냄
- 조건이나 글꼴색 지정 시 대괄호 안에 입력함
- 사용자 지정 서식 코드
    - 숫자 서식
        - # : 유효한 자릿수만 표시하고, 유효하지 않은 0은 표시하지 않음
        - 0 : 유효하지 않은 자릿수를 0으로 표시함
        - ? : 유효하지 않은 자릿수에 0대신 공백을 입력하고, 소수점을 기준으로 정렬함
        - . : 천 단위 구분 기호 표시 -> 표시형식 맨 끝에 표시하면 할 때마다 3자리씩 생략함
        - % : 숫자에 100을 곱한 다음 %을 붙임
        - DNNUM : 숫자를 한자 및 한글, 한자/한글로 표시
    - 문자 서식
        - @ : 문자 데이터의 표시 위치 지정
        - * : *기호 다음에 있는 특정 문자를 셀의 너비만큼 반복해 채움
        - _ : 셀에 입력된 데이터의 오른쪽 긑에 하나의 공백이 생김
    - 날짜 서식
        - 월
            - mmm : Jan ~ Dec로 표시
            - mmmm : January ~ December로 표시
        - 요일
            - ddd : Sun ~ Sat로 표시
            - dddd : Sunday ~ Saturday로 표시
    - 시간 서식
        - 시간
            - hh : 00 ~ 23으로 표시
            - [h] : 경과된 시간 표시
        - 분
            - mm : 00 ~ 59로 표시
            - [m] : 경과된 분 표시
        - 초
            - ss : 00 ~ 59로 표시
            - [s] : 경과된 초 표시
        
112. 조건부 서식
- 규칙에 만족하는 셀에만 셀 서식 적용
- 조건부 서식의 규칙을 수식으로 입력할 경우 수식 앞에 반드시 등호(=)를 입력해야함
- 수식의 결과는 참 또는 거짓이 나오도록 작성해야함
- 워크시트의 특정 셀을 이용해 규칙을 작성할 수 있고, 규칙 작성 시 마우스로 셀을 클릭하면 절대 참조로 지정됨
- 셀의 값이 변경되어 규칙을 만족하지 않으면 적용된 서식이 해제됨
- 셀에 입력된 값에 따라 데이터 막대, 색조, 아이콘 등을 표시할 수 있음
- 규칙별로 다른 서식을 적용할 수 있음
- 둘 이상의 조건부 서식이 참일 경우 두 규칙에 지정된 서식이 모두 적용되지만, 서식이 충돌할 경우 우선 순위가 높은 규칙의 서식이 적용됨
- 규칙에 맞는 데이터가 있는 행 전체에 서식을 지정할 때는 수식 입력시 열 이름 앞에 $를, 열 전체에 서식을 지정할 때는 행 번호 앞에 $를 붙임
- 다른 통합 문서를 참조해 조건을 지정할 수 없음

113. 수식의 개념
- 수식 : 워크시트에 입력된 데이터를 계산하거나 분석하기 위한 식
- 등호나 '+', '-' 기호로 시작함
- 문자열이 수식에 사용될 때는 큰따옴표로 묶어야함
- 수식을 선택한 영역 전체에 한 번에 입력하려면 수식을 입력한 후 ctrl+enter를 누름
- ctrl+~ 를 누르면 워크시트에 입력된 수식을 모두 볼 수 있음
- 수식을 입력 후 F9을 누르면 화면에 표시되는 값이 상수로 변환되어 수식 대신 셀에 입력됨

114. 오류 메시지
- ##### : 셀에 셀 너비보다 큰 수치 데이터나 음수의 날짜나 시간이 있을 때
- #DIV/0! : 피제수가 빈 셀이나 0이 있는 셀을 참조할 때
- #NAME? : 인식할 수 없는 텍스트를 수식에 사용했을 때
- #NULL! : 교차하지 않는 두 영역의 교점을 지정했을 때
- #NUM! : 표현할 수 있는 숫자의 범위를 벗어났을 때
- #REF! : 셀 참조가 유효하지 않을 때
- #VALUE! : 잘못된 인수나 피연산자를 사용할 때, 수식 자동 고침 기능으로 수식을 고칠 수 없을 때

115. 셀 참조
- 상대 참조 : 수식을 입력한 셀의 위치가 변동되면 참조가 상대적으로 변경됨(A1)
- 절대 참조 : 수식을 입력한 셀의 위치와 관계없이 고정된 주소로, 참조가 변경되지 않음($A$1)
- 혼합 참조
    - 열 고정 혼합 참조 : 열만 절대 참조가 적용됨($A1)
    - 행 고정 혼합 참조 : 행만 절대 참조가 적용됨(A$1)
- 다른 워크시트의 셀 참조
    - 다른 워크시트에 있는 셀의 데이터를 참조할 경우 시트 이름과 셀 주소 사이를 느낌표로 구분함(Sheet!A5)
    - 시트 이름에 한글, 영어 외의 문자가 있을 경우 작은따옴표로 묶음
- 3차원 참조
    - 여러 개 시트의 동일한 셀 주소, 또는 동일한 셀 범위에 대한 참조
- 다른 통합 문서의 셀 참조
    - 다른 통합 문서에 있는 셀의 데이터를 참조할 경우 통합 문서의 이름을 대괄호로 묶음

116. 이름 정의
- 자주 사용하는 셀이나 셀 범위에 이름을 지정하는 것으로 수식이나 함수에서 주소 대신 이름을 참조해 사용함
- 정의된 이름은 참조 시 절대 참조 방식으로 사용됨
- 이름 상자의 화살표 단추를 누르고 정의된 이름 중 하나를 클릭하면 해당 셀 또는 셀 범위가 선택됨
- 이름 작성 규칙
    - 첫 문자는 만드시 문자나 밑줄 또는 역슬래시
    - 통합 문서내 동일한 이름 중복 불가
    - 대소문자 구분하지 않으며 최대 255자까지
    - 셀 주소 형식으로 이름 지정 불가

117. 통계 함수
- AVERAGE(인수1, 인수2,...) : 인수들의 평균 반환
- AVERAGEA(인수1, 인수2,...) : 인수들의 평균 반환 -> 숫자가 아닌 셀도 인수로 사용
- AVERAGEIF(조건이 적용될 범위, 조건, 평균을 구할 범위) : 조건이 적용될 범위에서 조건에 맞는 셀을 찾아 평균을 구할 범위 중 같은 행에 있는 값들의 평균값 반환
- AVERAGEIFS(평균을 구할 범위, 조건1이 적용될 범위, 조건1, 조건2가 적용될 범위, 조건2,...) : 여러 개의 조건이 적용될 범위에서 여러 개의 조건에 맞는 셀을 찾아 평균을 구할 범위 중 같은 행에 있는 값을의 평균갑 반환
- MAX(인수1, 인수2,..) : 인수들 중 가장 큰 값 반환
- MAXA(인수1, 인수2,...) : 인수 중 가장 큰 값 반환 -> 숫자는 물론 빈 셀, 논리값, 숫자로 표시된 텍스트 등도 인수로 사용
- MIN(인수1, 인수2,...) : 인수들 중 가장 작은 값 반환
- MINA(인수1, 인수2,...) : 인수 중 가장 작은 값 반환 -> 숫자는 물론 빈 셀, 논리값, 숫자로 표시된 텍스트 등도 인수로 사용
- COUNT(인수1, 인수2,...) : 인수들 중 숫자가 있는 셀의 개수 반환
- COUNTA(인수1, 인수2,...) : 인수들 중 자료가 입력되어 있는 셀의 개수 반환
- COUNTBLANK(범위) : 범위 중 자료가 없는 셀의 개수 반환
- COUNTIF(범위, 조건) : 지정된 범위에서 조건에 맞는 셀의 개수 반환
- COUNTIFS(조건1이 적용될 범위, 조건1, 조건2가 적용될 범위, 조건2,...) : 여러 개의 조건이 적용될 범위에서 여러 개의 조건에 맞는 셀을 찾아 개수 반환
- LARGE(범위, n번째) : 범위 중 n번째로 큰 값 반환
- SMALL(범위, n번째) : 범위 중 n번째로 작은 값 반환
- RANK.EQ(인수, 범위, 옵션) : 지정된 범위 안에서 인수의 순위 반환 -> 동일한 값들은 동일하지 않을 경우 나올 수 있는 순위들 중 가장 높은 순위를 동일하게 반환
    - 옵션
        - 0 또는 생략 : 내림차순 기준 순위 부여
        - 0 이외 값 : 오름차순 기준 순위 부여
- VAR.S(인수1, 인수2,...) : 인수로 주어진 숫자들의 표본 분산값 반환
- STDEV.S(인수1, 인수2,...) : 인수로 주어진 숫자들의 표본 표준편차값 반환
- MEDIAN(인수1, 인수2,...) : 인수들의 중간값 반환
- MODE.SNGL(인수1, 인수2,...) : 인수 중 가장 빈도수가 높은 값 반환
- FREQUENCY(배열1, 배열2) : 배열2의 범위에 대한 배열1 요소들의 빈도수 반환
- GEOMEAN(인수1, 인수2,...) ㅣ 인수로 주어진 숫자들의 기하 평균 반환
- HARMEAN(인수1, 인수2, ...) : 인수로 주어진 숫자들의 조화 평균 반환
- PERCENTILE(범위, 인수) : 범위에서 인수 번째 백분위수 값 반환

118. 수학/삼각 함수
- SUM(인수1, 인수2,...) : 인수들의 합 반환
- SUMIF(조건이 적용될 범위, 조건, 합계를 구할 범위) : 조건에 맞는 셀을 찾아 합 반환
- SUMIFS(합계를 구할 범위, 조건1이 적용될 범위, 조건1, 조건2가 적용될 범위, 조건2,...) : 여러 개의 조건이 적용될 범위에서 여러 개의 조건에 맞는 셀을 찾아 합계를 구할 범위 중 같은 행에 있는 값들의 합 반환
- ROUND(인수, 반올림 자릿수) : 인수에 대해 지정한 밥ㄴ올림 자릿수로 반올림함
- ROUNDUP(인수, 올림 자릿수) : 인수에 대해 지정한 올림 자릿수로 올림
- ROUNDDOWN(인수, 내림 자릿수) : 인수에 대해 지정한 내림 자릿수로 내림
- SUMPRODUCT(배열1, 배열2,...) : 배열1과 배열2의 개별 요소들끼리 곱한 결과를 모두 더한 값 반환
- MIDETERM(배열) : 배열의 행렬식 반환
- MINVERSE(배열) : 배열의 역행렬 반환
- MIMULT(배열1, 배열2) : 배열1과 배열2의 행렬 곱 반환
- RAND() : 0과 1 사이 난수 반환
- RANDBETWEEN(인수1, 인수2) : 지정한 두 수 사이 난수 반환
- ABS(인수) : 인수의 절대값 반환
- INT(인수) : 인수보다 크지 않은 정수값 반환
- MOD(인수1, 인수2) : 인수1을 인수2로 나눈 나머지값 반환
- FACT(인수) : 인수의 계승 값 반환
- SQRT(인수) : 인수의 양의 제곱근 반환 -> 인수가 음수면 에러 발생
- POWER(인수, 제곱값) : 인수를 제곱값만큼 거듭 곱한 값 반환
- TRUNC(인수, 자릿수) : 인수에 대해 자릿수 미만의 수치를 버린 값을 반환
- PRODUCT(인수1, 인수2,...) : 인수를 모두 곱한 값 반환
- SIGN(인수) : 인수의 부호값 반환 -> 양수면 1, 0이면 0, 음수면 -1
- PI() : 파이를 15자리까지 계산
- EXP(인수) : e를 인수만큼 거듭제곱한 값 반환
- QUOTIEN(인수1, 인수2) : 인수1을 인수2로 나누어 몫에 해당하는 정수 부분만을 반환

126. 배열 수식/배열 상수
- 배열 수식
    - 배열 인수라는 2개 이상의 값에 의해 이루어짐
    - 배열 인수 각각은 동일한 개수의 행과 열을 가져야 함

    - 수식 입력줄이 활성화되면 배열 수식의 중괄호는 나타나지 않음

    - 배열 수식에 사용되는 배열 인수

    - 배열 상수로 사용할 수 없는 값 : $, 괄호, %, 길이가 다른 행이나 열, 셀 참조 등

128. 차트의 개요
- 차트 이용시 데이터의 추세나 유형 등을 쉽게 이해 가능, 많은 양의 데이터를 간결하게 요약 가능
- 차트 작성시 반드시 원본 데이터가 있어야 함
- 원본 데이터가 바뀌면 차트 모양도 바뀜
- 데이터가 입력된 셀 중 하나를 선택한 상태에서 차트를 만들면 해당 셀을 둘러싼 모든 셀의 데이터가 차트에 표시됨
- 2차원과 3차원 차트로 구분됨
- 차트만 별도로 표시할 수 있는 차트 시트 생성 가능
- 기본 차트는 묶은 세로 막대형 -> 사용자가 변경 가능
- 데이터 범위 지정 후 F11 누르면 별도의 차트 시트에 기본 차트가 작성됨
- alt+F11을 누르면 데이터가 있는 워크시트에 기본 차트가 작성됨

130. 차트의 편집
- 원본 데이터 변경 : 데이터 범위 변경, 데이터 계열 추가/제거 및 계열 방향 변경 -> '데이터 원본 선택' 대화상자에서 지정
- 데이터 추가 : 추가할 데이터 범위를 복사해 차트 영역 선택 후 붙여넣기
- 특정 셀의 텍스트를 차트 제목으로 연결

131. 추세선
- 특정한 데이터 계역의 변화 추이 파악 위해 표시하는 선
- 추세선 표시 : 추세선을 표시할 데이터 계열 선택 후 차트디자인->차트레이아웃->차트요소추기->추세선에서 적용할 추세선 선택
- 종류 : 선형, 로그, 다항식, 거듭제곱, 지수, 이동 평균
- 3차원, 방사형, 원형, 도넛형, 표면형 차트에는 추세선 불가
- 추세선이 추가된 차트를 3차원으로 변경하면 추세선 제거됨
- 하나의 데이터 계열에 두 개 이상의 추세선 표시 가능

132. 오차 막대
- 데이터 계열의 오차량을 그림으로 나타낸 것
- 고정값, 백분율, 표준 편차, 표준 오차 등으로 표시 가능
- 3차원 차트에는 표시 불가
- 세로 오차 막대 적용 가능 차트 : 영역형, 세로 막대형, 꺽은선형, 분산형, 거품형
- 세로 오차 막대, 가로 오차 막대 적용 가능 차트 : 분산형,ㄱ ㅓ품현

133. 용도별 차트의 종류
- 세로/가로 막대형 : 각 항목 간의 값을 막대의 길이로 비교/분석
- 꺾은선형 : 일정 기간 동안의 데이터 변화 추이 확인
- 원형 : 전체 항목의 합에 대한 각 항목의 비율, 중요한 요소 강조
- 분산형 : X/Y 좌표로 이루어진 한 개의 계열로 두 개의 숫자 그룹 나타냄, 데이터의 불규칙한 간격이나 묶음 보여줌
- 영역형 : 시간에 따른 각 값의 변화량 비교
- 도넛형 : 전체에 대한 각 부분의 관계를 비율로 나타내 각 부분 비교
- 방사형 : 많은 데이터 계열의 집합적인 값 나타냄
- 표면형 : 두 개의 데이터 집합에서 최적의 조합 찾을 때
- 거품형 : 분산형의 한 종류 -> 데이터 계열값이 세 개인 경우
- 주식형 : 주가의 흐름을 파악
- 트리맵 : 계층 간의 상대적 크기 비교
- 선버스트 : 계층 간의 관계 비교
- 히스토그램 : 특정 범위 그룹화해 그룹별 데이터 분포 표시

150. 피벗 테이블
- 많은 양의 데이터를 한눈에 쉽게 파악할 수 있도록 요약/분석해 보여주는 도구
- 엑셀 목록, 데이터베이스, 외부 데이터, 다른 피벗 테이들 등의 데이터 사용 가능
- 필드별로 다양한 조건 지정 가능
- 그룹별로 데이터 집계 가능
- 원본 데이터가 변경되면 새로고침을 이용해 피벗 데이틀의 데이터도 변경 가능
- 사용자가 피벗 테이블에 새로운 필드 추가 가능

151. 피벗 차트 보고서
- 피벗 테이블의 데이터를 이용해 작성한 파트
- 피벗 테이블에서 항목이나 필드에 변화를 주면 피벗 차트도 변경됨
- 피벗 차트에 변화를 주면 피벗 테이블도 변경됨
- 피벗 차트를 추가하면 피벗 테이블이 있는 워크시트에 삽입됨

152. 피벗 테이블의 그룹화

- 문자, 숫자, 날자, 시간 등 모든 필드에서 사용 가능

153. 시나리오
- 다양한 상황과 변수에 따른 여러 가지 결과값의 변화를 가상의 상황을 통해 예측/분석하는 도구
- 이자율, 손익 분기점, 주가 분석 등에 많이 사용됨
- 시나리오 작성시 현재 작업하는 워크시트 왼족에 새 워크시트 삽입하고 그 시트에 시나리오 보고서 표시
- 여러 시나리오 비교 위해 시나리오를 피벗 테이블로 요약 가능
- 시나리오 요약 보고서나 시나리오 피벗 테이블 보고서 만들때는 반드시 결과 셀을 지정해야 함
- 시나리오 병합을 통해 다른 통합 문서나 다른 워크시트에 저장ㄷ왼 시나리오를 가져올 수 있음
- 시나리오를 삭제해도 이미 작성된 시나리오 요약 보고서는 삭제되지 않음
- 시나리오가 작성된 원본 데이터를 변경해도 이미 작성된 시나리오 요약 보고서에는 반영되지 않음

154. 목표값 찾기
- 수식에서 원하는 결과(목표)값은 알고 있지만 그 결과값을 계산하기 위해 필요한 입력값을 모를 경우 사용
- 데이터->예측->가상 분석->목표값 찾기
- 수식 셀 : 결과값이 출력되는 셀 주고 -> 반드시 수식이 있어야 함
- 찾는 값 : 목표로 하는 값 직접 입력
- 값을 바굴 셀 : 목표값을 만들기 위해 변경되는 값이 들어 있는 셀 주소로

155. 데이터 표
- 특정 값의 변화에 따른 결과값의 변화 과정을 표의 형태로 표시해주는 도구
- 결과값은 반드시 변화하는 특정 값을 포함한 수식으로 작성되어야 함
- 변화하는 값과 수식이 입력된 부분을 모두 포함되도록 범위 설정 후 데이터 표 실행
- 행 입력 셀 : 변화되는 값이 항 행에 있을 때 변화되는 셀의 주소 지정
- 열 입력 셀 : 변화되는 값이 한 열에 있을 때 변화되는 셀의 주소 지정
