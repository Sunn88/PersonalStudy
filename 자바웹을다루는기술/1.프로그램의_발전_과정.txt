1. 클라이언트 PC 기반 프로그램
- 클라이언트 기반 프로그램의 문제점 : 기능이 자주 변경되는 프로그램이라면 수시로 사용자 PC마다 프로그램을 업데이트하거나 새로 설치해야함
- 클라이언트-서버 기반 프로그램 : 클라이언트 PC 기반 프로그램의 문제점을 보완해 나온 것

2. 클라이언트-서버 기반 프로그램 동작 방식
- 기존 클라이언트가 수행하는 모든 기능을 서버에서 수행함

3. 웹 기반 프로그램 동작 방식
- 클라이언트는 자신이 사용하는 클라이언트 프로그램을 직접 설치하는 것이 아니라 사용자 컴퓨터의 웹 브라우저를 통해 화면에 해당되는 HTML 문서를 서버에 요청함
- 서버에서는 요청 받은 HTML 문서를 브라우저에 전송해 해당 기능을 담당하는 화면을 보여줌
