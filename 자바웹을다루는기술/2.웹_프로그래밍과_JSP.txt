1.웹 프로그래밍의 기본
- JSP 등장 이전 : 아파치 같은 웹 서버를 이용해 웹 페이지들을 브라우저에 표시 -> 정적(static) 웹 프로그래밍
    - 웹 서버에 미리 보여줄 HTML 페이지, CSS, 이미지, 자바스크립트 파일을 저장해 놓고 요청시 그대로 전달하는 방식
    - 페이지가 변경되지 않는 한 고정된 웹 페이지를 보게 됨
    - 요즘은 잘 사용되지 않음
    - 정적 웹 프로그래밍 구성 요소는 JSP에서도 여전히 화면 디자인과 기능 처리를 담당함
- 정적 웹 프로그래밍 구성 요소
    - 웹 서버 : 각 클라이언트에게 서비스를 제공하는 컴퓨터
    - 클라이언트 : 네트워크로 서버에 접속한 수 서버로부터 서비스를 제공받는 컴퓨터
