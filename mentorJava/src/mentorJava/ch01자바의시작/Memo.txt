자바의 특징
 - 객체지향 언어 : 상속, 캡슐화, 다양성
 - 운영체제(플랫폼)에 독립적 : 자바 가상머신(JVM)을 통해 어떤 운영체제에서도 동일하게 실행 가능
 - 가비지 컬렉션(Garbage Collection)지원 : 개발자가 메모리를 관리하지 않아도 되는 메모리 회수 기법 -> 자동으로 메모리 관리해 코드가 간결해짐
 - 동적 : 프로그래밍 실행시 적절한 시점에 실행에 필요한 라이브러리들을 연결
 - 스레드 지원 : 스레드 관련 라이브러리를 제공 -> 스레드 활용해 프로그램을 작성 -> 네트워크 프로그래밍이나 복잡한 작업을 효과적으로 처리
 - 네트워크 프로그래밍과 분산처리를 지원 : 다양한 네트워크 관련 API가 제공됨, 분산 환경 프로그래밍에 적합함
 
개발환경 준비
