1. 조건문
 - 조건문 : 특정한 조건을 만족하면 수행할 작업과, 만족하지 않을 때 수행할 작업을 정해 놓은 제어문
 1) if문
     if(조건식) {
     	(실행부분)
     }
 - 조건식 : 식의 결과가 true.false로 구분될 수 있는 문장
 - 실행부분 : 조건식의 결과가 true면 실행할 문장
2) if ~ else문 : if문의 구조에서 조건식이 false일 경우 실행되는 부분을 추가
    if(조건식){
    	(실행부분1)
    }else{
    	(실행부분2)
    }
 - 실행부분1 : 조건식의 결과가 true면 실행할 문장
 - 실행부분2 : 조건식의 결과가 false면 실행할 문장
3) if ~ else if문 : 조건이 여러 가지일 경우 조건에 따라 각각 다른 작업을 수행하고 싶을 때 사용
	if(조건1){
		실행1
	}else if(조건2){
		실행2
	}else if(조건3){
		실행3
	}else{
		실행4
	}
