1. 컴퓨터 발전과 4차 산업혁명(1)
- 학습목표
    - 인공지능과 빅데이터를 공부하기 전에 컴퓨터 발전과 4차 산업 혁명의 의미를 이해
- 최초의 컴퓨터는 전산실의 대형 컴퓨터
- 데이터를 저장하기 위한 메모리와 운영을 위한 프로그램(Operating System) 주변장치로 대형 컴퓨터 저장 장치인 디스크와 테이프가 있었음
- 데이터와 정보의 차이
    - 작은 데이터들이 모여 빅데이터가 되고 고객 관리의 정보로 활용
    - Garbage Colleection
- 내가 가진 데이터의 합이 나를 말한다

2. 컴퓨터 발전과 4차 산업혁명(2)
- ICT 기반의 소프트웨어 혁신 과정
- 4차 산업혁명에서 활용하는 기술들
- 다양성 존중
- 유연한 생각
- 스스로 회복

3. 컴퓨터 발전과 4차 산업혁명(3)
- 4차 산업혁명 : 정보통신기술의 융합으로 이루어 낸 혁명 시대

4. 빅데이터와 인공지능의 개념(1)
- 학슴목표
    - 빅데이터와 인공지능에 관해 구체적인 개념을 파악하고 상호 관계를 이해
- 인공지능(AI) : 기계가 인간처럼 사고, 행동할 수 있도록 하는 기술
    - 인간의 지능을 컴퓨터 프로그램으로 구현
    - 학습, 추론, 문제 해결, 인지 기능을 수행하도록 하는 기술
    - 주요 기술 : 머신 러닝, 딥 러닝, 로보틱스, 자연어 처리 컴퓨터 비전
- 인공지능의 중요성
    - 비즈니스와 일상생활의 혁신
    - 기술 및 사회 전반에 걸친 변화
    - 공장 자동화 및 인간 노동의 보조와 자원
- AI 응용분야 : 자율주행차량, 의료 진단, 추천 시스템, Smart Assistant, 문서작성, 번역, 길 안내 상품 구매 서비스 등

5. 빅데이터와 인공지능의 개념(2)
- 인공지능의 발전과 연구 분야
    - 제1기(1956~1973) : 기계학습, 인지과학, 로봇공학 등 AI의 기본 개념과 방법론 개발
    - 제2기(1974~1987) : AI 한계와 어려움이 드러나고 연구가 위축
    - 제3기(1988~현재) : 컴퓨터 성능의 향상과 새로운 기술의 등장, AI 연구가 다시 활발해지고, 부분적으로 실용화된 상황
- 빅데이터 : 방대한 양의 데이터를 분석해 새로운 패턴이라는 의미를 발견하는 방법론
- Machine Learning : 컴퓨터가 데이터를 학습해 스스로 문제를 해결하는 방법론
- Deep Learning : 인공 신경망을 이용한 머신 러닝 방법론

6. 데이터 과학과 인공지능의 관계(1)
- 학습 목표
    - 인공지능의 기본은 데이터로부터 출발
    - 데이터와 인공지능의 관계와 상호 역할에 관해 정리
- Data Science
    - 데이터로부터 지식과 insight를 추출하는 학문 분야
    - 다양한 분야의 기술과 방법론의 융합
    - 통계학, 컴퓨터 공학, 정보과학 및 다양한 분야에 필요한 전문 지식
- 데이터 과학의 주요 기술
    - Data Mining : 데이터에 숨겨져 있거나 이전에 알려지지 않은 패턴을 추출하고 쓸모없는 데이터를 찾아내는 기술
    - Maching Learning : 컴퓨터가 데이터를 학습해 스스로 문제를 해결하는 방법론
    - Big Data 분석 및 Data의 시각화
- 데이터 과학의 중요성
    - 데이터를 기반으로 한 의사 결정
        - 기업과 조직에서 데이터를 기반으로 한 의사 결정 증가
        - 객관적 분석을 통한 전략 개발
    - 혁신과 경쟁력 강화
        - 데이터를 통한 새로운 비즈니스 모델 및 서비스 개발
        - 시장에서의 경쟁령 강화 및 우위 확보
    - 사회적 영향
        - 공공분야의 효율성 향상, 의료, 교통, 에너지 등 다양한 분야에서의 업무 개선 및 자동화

7. 데이터 과학과 인공지능의 관계(2)
- 빅데이터 활용법
    - 1단계 : 필요한 데이터를 검색하고 준비
    - 2단계 : 기업의 비즈니스 요구 사항을 확인
    - 3단계 : 모델링 검증 작업
    - 4단계 : 데이터를 분석할 수 있는 모델 구축
    - 5단계 : 실전에 적합한지 확인하고 거증
- 데이터 처리의 변화
    - 데이터 형태 : 특정 형식, 특정 일시 -> 형식, 일시 무관
    - 데이터 처리 속도 : 일괄처리(Batch Processing) -> 실시간 처리(Real Time)
    - 데이터 처리 목적 : 과거 분석 -> 최적화 및 예측
    - 데이터 처리 비용 : 정부 지원 -> 개별 기업 차원, 개인 부담 등
