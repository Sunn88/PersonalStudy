답변 시 유의할 세 가지 포인트
- 정확한 문법 사용
    - 의문사 잘 확인하기
    - 시제 유의하기
- 현장감 살려 말하기
    - 중요 키워드(의문사, 답변) 크게 말하기
    - 일정한 리듬으로 답변하기
    - 영어 면접을 보는 것처럼 답변하기
- 논리적으로 말하기
    - 답변이 질문의 의도에 부합하는지 확인
    - 상대방이 내 답변에 논리적으로 납득할지 확인
    - 답변을 통째로 외우지 않기

문제 유형별 답변 만들기
- 5번 : 두 개의 질문을 묻는 유형
    - 정확한 문법 사용 : 의문사, 시제 유의
    - 현장감 살려 말하기
        - 영어 면접을 보는 것처럼 말하기
        - 중요 키워드 크게 말하기
        - 일정한 리듬으로 답변하기
    - 의문사 체크하기
    - 주어 선정하기
    - 고득점 포인트
        - 의문사 별 답변 키워드 준비(2 ~ 3가지) : How often, where, when 등
        - 연습 문제를 많이 접하기 : 의문문 문장 구조에 익숙해지기
        - 같은 문제를 반복 연습 : 자연스러운 속도로 답변할 수 있을 때까지
- 6번 : 이유를 추가로 묻는 유형
    - 이유 문장을 만드는 연습 : 채점 비율 높음
    - 이유 문장 대표 구문
        - ★가능성 : I can + v
        - ★성질/상태 : It is 형용사 to + V
        - ★존재 유무 : Thers is + N
        - 습관 : Nowadays, I usually + V
        - 필요성 : I need to + V / I should + V / I don`t have to + V
        - 선호도 : I`m (not) interested in + N / V-ing
        - 능력 : I`m (not) good at + N / V-ing
    - 고득점 포인트
        - 이유 문장 대표 구문 확실히 학습하기
        - 한 문제에 다양한 이유를 만들기
        - 만든 이유 문장을 따로 정리해두기
- 7번 : 의견을 설명하는 유형
    - 답변 네 가지 패턴
        - A : 입장->이유->이유2
        - A+ : 입장->이유->(이유2+보강 설명)
        - B : 입장->이유+보강 설명
        - B+ : 입장->(이유 + 보강 설명)->이유2
        - 시간이 남으면 마무리 문장 추가
