02. C언어 소개
- 탄생
    - 기존 운영체제들은 하드웨어 종속적인 언어를 사용함
    - 하드웨어가 변경되어도 프로그램을 다시 작성하지 않아도 되는 운영체제 필요
- 특징
    - 구조화된 언어 : 일정한 단위로 명령들을 그룹으로 묶는 방법 제공 -> 유지보수/재사용 편리
    - 이식성이 높음 : C언어로 만든 프로그램을 거의 수정하지 않고 다른 하드웨어 또는 운영체제로 가져가도 잘 동작함
        - 하드웨어 종속적인 부분은 런타인 라이브러로(Runtime Library)로 제공함
    - 확장성이 좋고, 표현법이 다양하 자기만의 표현법을 개발할 수 있음
        - 사용할 확률이 높은 문법만 최소한 유지, 사용 빈도가 낮은 문법은 스스로 구현해 언어의 기능 확장 가능
        - 상대적으로 어렵게 느껴질 수 있음
    - 생산성이 높음 : 스스로 개발 인프라를 구축하기 때문에 경력이 쌓일수록 개발속도가 빨라짐
