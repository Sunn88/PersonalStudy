1) C언어와 Java언어의 개요
- C언어
    - 1972년 미국 벨 연구소의 데니스 리치에 의해 개발된 ㄴ시스템 기술용 언어
    - 유닉스(UNIX) 운영체제 개발에 사용할 목적으로 개발됨
    - 특징
        - 논리적이며 구조적인 시스템 프로그래밍 언어
        - 하드웨어 제어가 가능해 프로그램 이식성이 높음
        - 간략한 문법 표현으로 함축적인 프로그램 작성 용이
        - 효율성과 유연성을 갖춘 저급 언어 특성을 가진 고급 언어
2) C언어의 기본 구조
- 소스코드는 파일명.c로 작성됨
- 함수 중심으로 이루어짐
- 도입 부분, main() 함수 부분, 사용자 정의(호출) 함수
- 번역은 소스코드의 상단부터 진행되지만, 번역 후 실제 실행은 main()함수에서부터 실행됨
3) Java언어의 기본 구조
- 소스코드는 클래스명.java로 작성됨
- 클래스명은 반드시 대문자로 시작해야 함
- 클래스 중심 -> 클래스 내의 메소드를 통해 기능 구현
4) C언어와 Java언어의 기본 소스코드 비교
5) C언어와 Java언어의 프로그램 구성요소
- C언어
    - 예약어(reserved word) : 키워드
        - 자료형관련, 기억관련, 제어관련, 기타
        - int, char, static, if~else, for, while, include 등
    - 명칭(identifier) : 식별자
        - 변수명, 배열명, 함수명, 매크로명
    - 상수(constant)
        - 정수상수, 실수상수, 문자상수, 문자열상수
    - 연산자(operator)
    - 설명문(comment) : 주석
        - 비실행문
        - 한 줄 주석
        - 여러 줄 주석
- Java언어
    - 예약어(reserved word) : 키워드
        - 자료형관련, 기억관련, 제어관련, 기타
        - int, char, static, if~else, for, while, import 등
    - 명칭(identified) : 식별자
        - 클래스명, 메소드명, 변수명, 배열명, 함수명
        - (관례)클래스 이름의 경우 대문자로 시작, 메소드나 필드/변수의 이름은 소문자로 시작
    - 상수(constant)
        - 논리상수, 정수상수, 실수상수, 문자상수, 문자열상수
    - 연산자(operator)
    - 설명문(comment) : 주석
        - 비실행문
        - 한 줄 주석
        - 여러 줄 주석
6) C언어 기본문법 구조와 printf() 함수
- printf() 함수
    - 기능 : 콘솔 화면에 주어진 출력 양식으로 자료(상수)를 출력하는 함수
        - "문자열상수" 또는 인수의 값을 출력
    - 형식 : #include <stdio.h>*
             printf("출력형식", 출력대상1, 출력대상2, ...);
- 출력형식 변환문자
    - %d : 10진 정수로 변환해 출력
    - %f : 부동소수점을 실수로 변환해 출력
    - %c : 한 문자로 변환해 출력
    - %s : 문자열로 변환해 출력
    - %o(8진 정수), %x(16진 정수), %u(부호 없는 10진 정수), %e(지수)
* 전처리기(preprocessor) : 시스템이 제공하는 소스코드(라이브러리)를 미리 포함한 후 컴파일을 진행하도록 사전 작업을 처리해 주는 시스템 프로그램
7) Java의 System.out.print()와 System.out.println()
- java.lang 패키지에서 제공하는 표준출력문(메소드)
- 괄호 사이에 출력하고자 하는 변수나 문자열이 올 수 있음
- 자동 줄바꿈 유무의 차이
