1) 스크립트 언어
- 스크립트 언어(Script Language)의 개념
    - 소스 코드를 컴파일 과정을 거치지 않고 실행할 수 있는 프로그래밍 언어
    - 스크립트 언어에 내장된 번역기에 의해 번역되어 실햏ㅇ됨
    - 실행 단계에서 구문을 분석함
    - Scrapy : 파이썬 기반의 웹크롤러 프레임워크로서 가볍고 빠르고 확장성이 좋음
- 스크립트 언어의 종류
    - 서버 측 스크립트 언어
        - ASP(Active Server Page) : 서버 측에서 동적으로 수행되는 페이지를 만들기 위한 언어로 Windows 계열의 운영체제에서 실행 가능함
        - JSP(Java Server Page) : Java를 기반으로 하고 서버 측에서 동적으로 수행하는 페이지를 만드는 언어
        - PHP(Professional Hypertext Preprocessor)
            - 소스 코드가 HTML 파일에 포함되는 언어
            - 데이터베이스와의 연동이 매우 용이함
            - LINUX, UNIX, Windows 등의 다양한 운영체제에서 사용 가능함
            - PHP 연산자
                - 산술 연산자 : +, -, *, /, %, **
                - 할당 연산자 : =, +=, -=, *=, /=, %=
                - 중가/감소 연산자 : ++, --
                - 관계 연산자 : ==, ===, !=, <>, !==, >, <, >=, <=
                - 논리 연산자 :  and, or, xor, &&, ||, !
        - 파이썬(Python)
            - 인터프리터 방식의 객체지향 언어
            - 실행 시점에 데이터 타입을 결정하는 동적 타이핑 기능을 가짐
    - 클라이언트 측 스크립트 언어
        - JavaScript
            - HTML 문서에서 HTML이나 CSS로 표현하기 어렵거나 불가능한 작업을 수행하기 위해 개발됨
            - 소스 코드가 HTML 문서에 포함되어 있음
            - 클래스가 존재하지 않으며 변수 선언도 필요 없음(ESS 버전 부터 지원)
            - 사용자의 웹 브라우저에서 직접 번역되고 실행됨
        - VBScript
            - 마이크로소프트가 개발한 액티브 스크립트 언어
            - VBScript의 구문은 비주얼 베이직(Visual Basic) 프로그래밍 언어를 일부 반영함
2) 파이썬(Python)
- 파이썬의 개요
    - 1991년 귀도 반 로섬이 개발한 고급 프로그래밍 언어
    - 플랫폼에 독립적이고 인터프리터식, 객체지향적, 동적 타이핑(Dynamically Typed) 대화형 언어임
    - 매우 쉬운 문법 구조로 초보자들도 쉽게 배울 수 있음
- 파이썬 변수명 작성 규칙
    - 영문 대소문자, 숫자, '_'를 혼용해 사용할 수 있음
    - 첫 글자는 영문자나 '_'로 시작해야 함
    - 영문자는 대소문자는 구분함
    - 공백을 포함할 수 없음
    - 예약어(Reserved Word)를 사용할 수 없음
- 문자열 추출하기
    - 하나의 문자를 추출하려면 추출하려는 문자의 인덱스(0부터 시작)를 지정함
    - 역순으로 맨 오른쪽의 인덱스는 -1
- 파이썬의 range() 함수 : for 반복문과 함께 많이 사용되며, 주어진 인수로 0부터 연속된 정수를 리스트 객체로 반환하는 함수
- 파이썬의 함수 호출과 매개 변수 전달 : 파이썬의 cs() 함수는 정수를 전달받아 0부터 정수까지의 합을 누적하여 반환하도록 정의되어 있음
- 파이썬의 if~elif~else 조건문
- Scrapy
    - 파이썬 기반의 웹크롤러 프레임워크
    - 가볍고 빠르고 확장성이 좋음
- Python 표준 라이브러리
    - 라이브러리를 사용하려면 'import random'과 같이 선언하고, 사용할 때는 마침표(.)로 구분하여 'random.choice()'와 같이 사용함
    - Java는 표준라이브러리에 여러 패키지가 포함되어 있으나, Python은 표준 라이브러리라는 하나의 패키지만 존재함
    - 라이브러리를 사용할 떄 패키지가 아닌 각 클래스를 호출하여 사용함
        - os
            - 운영체제와 상호 작용하기 위한 기능을 제공함
            - getcwd(), chdir(), system()
        - re
            - 고급 문자열 처리를 위한 기능을 제공함
            - findall(), sub()
        - datetime
            - 날짜, 시간 조작을 위한 기능을 제공함
            - today(), date(), strftime()
        - math
            - 복잡한 수학 연산을 위한 기능을 제공함
            - cos(), log()
        - random
            - 무작위 선택을 위한 기능을 제공함
            - choice(), sample(), random(), randrange()
        - statistics
            - 통계값 산출을 위한 기능을 제공함
            - mean(), median(), variance()
        - 내장함수
            - Python에 기본적인 인터페이스로 import문이나 클래스명 없이도 사용할 수 있음
            - abs(), slice(), pow(), print()
- 딕셔너리(Dictionary)
    - 사전형 데이터를 의미하며, Key와 Value를 1대1로 대응시킨 형태로 하나의 Key에는 하나의 Value만이 대응됨
    - 리스트나 튜플처럼 순차적으로(Sequential) 해당 요솟값을 구하지 않고 Key를 통해 Value를 얻음
    - 키 값에 List, Set이 올 수 없고 키 값은 중복될 수 있음
