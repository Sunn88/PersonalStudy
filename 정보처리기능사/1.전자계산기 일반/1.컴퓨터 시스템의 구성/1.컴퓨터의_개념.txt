1) 컴퓨터의 개요
- 컴퓨터(EDFS; Electronic Data Processing System)
    - 입력된 자료를 프로그램이라는 명령 순서에 따라 처리
        - 자료(Data) : 관찰이나 측정을 통해 수집한 단순한 사실이나 결과값
            - 인간 또는 기계가 감지할 수 있도록 숫자, 문자, 기호 등을 이용해 형식화한 것
        - 정보(Information) : 의사 결정에 도움을 줄 수 있는 유용한 형태로 자료를 가공처리한 것
    - 그 결과를 사람이 알아볼 수 있도록 출력하는 전자(Eletronic) 처리(Data Processing) 시스템

2) 컴퓨터의 특징
- 대용량성 : 많은 양의 데이터를 처리 및 보관
- 범용성 : 컴퓨터를 여러 가지 용도로 사용
- 호환성 : 하나의 하드웨어나 소프트웨어를 컴퓨터의 기종에 관계없이 여러 컴퓨터에서 사용 가능
- 정확성 : 사용자의 요구 기능을 충족시키는 정도
- 신뢰성 : 주어진 환경에서 고장 없이 담당 기능을 원활하게 수행하는 정도
* GIGO(Garbage In Garbage Out) : 쓰레기가 들어가면 쓰레기가 나옴
    - 아무리 정확한 컴퓨터라도 사람이 잘못된 자료를 입력하면 컴퓨터도 잘못된 결과를 출력함
    - 컴퓨터의 수동성

3) 컴퓨터의 구성
- 하드웨어 : 컴퓨터 시스템을 구성하는 물리적인 부품
    - 중앙처리장치
        - 레지스터 : 임시 기억장소
        - 제어장치 : 명령 해독, 제어신호 전송
        - 연산장치 : 실제 연산 수행
    - 주변장치 = 입/출력장치 + 보조기억장치
- 소프트웨어 : 하드웨어를 사용하기 위한 각종 명령의 집합
    - 소프트웨어 = 시스템 소프트웨어 + 응용 소프트웨어
- 펌웨어 : 하드웨어적으로 구성된 소프트웨어
    - 주로 ROM에 반영구적으로 저장되어 하드웨어를 제어/관리
