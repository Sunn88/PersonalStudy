1) 입력장치
- 키보드 : 문자나 기호의 입력, 커서 이동 등의 작업
- 마우스 : 볼의 회전이나 빛의 반사를 감짛, 마우스 포인터의 움직임을 인식해 컴퓨터에 입력
- 스캐너  영상 정보에 빛을 쪼인 후 반사되는 빛의 차이를 감지해 디지털 그래픽 정보로 변환해줌
- 라이트 펜 : 빛을 인식할 수 있는 모니터의 특정 부분을 눌러 해당 점의 위치를 컴퓨터에 입력
- 광학 마크 판독기(OMR) : 컴퓨터용 수성 사인펜 표시에 빛을 비추어 표시 여부를 판독
- 광학 문자 판독기(OCR) : 특정 글골로 인쇄된 문자에 빛을 비춰 반사된 빛의 차이를 이용해 문자 판독
- 자기 잉크 문자 판독기(MICR) : 자성을 띤 특수 잉크로 인쇄된 문자나 기호를 판독 -> 수표나 어음
- 바코드 판독기(BCR) : 굵기가 서로 다른 선에 빛을 비춰 반사된 값을 코드화해 판독

2) 출력장치
- 모니터
- 프린터
- 플로터 : 용지의 크기에 제한 없이 고해상도 출력이 가능한 인쇄장치

3) 보조기억장치
- 주기억장치의 단점 보완
- 속도는 느리지만 용량이 큼
- 입/출력 겸용 장치
- 종류 : 자기 디스크, 자기 테이프, CD-ROM, 하드디스크
