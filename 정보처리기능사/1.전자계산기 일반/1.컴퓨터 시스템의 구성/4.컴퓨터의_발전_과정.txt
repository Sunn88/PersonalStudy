1) 컴퓨터의 기원


2) 컴퓨터의 세대별 특징
- 제1세대
    - 논리소자 : 진공관
    - 주기억장치 : 자기 드럼
    - 특징 : 기계어 사용, 하드웨어 중심, 일괄 처리 시스템
-제2세대
    - 논리소자 : 트랜지스터(TR)
    - 주기억장치 : 자기 코어
    - 특징 : 고급언어 개발, 운영체제 도입, 온라인 실시간 처리, 다중 프로그램
- 제3세대
    - 논리소자 : 집적회로(IC)
    - 주기억장치 : 집적회로
    - 특징 : 시분할 처리, 다중 처리, OCR/OMR/MICR/MIS 도입
- 제4세대
    - 논리소자 : 고밀도 집적회로(LSI)
    - 주기억장치 : 고밀도 집적회로
    - 특징 : 개인용 컴퓨터 개발, 마이크로프로세서 개발, 네트워크, 분산 처리
- 제5세대
    - 논리소자 : 초고밀도 집적회로(VLSI)
    - 주기억장치 : 초고밀도 집적회로
    - 특징 : 인터넷, 인공지능, 퍼지 이론, 패턴 인식, 전문가 시스템 등 신기술 개발
        - 퍼지 이론 : 불확실하고 애매한 자료를 컴퓨터에서 유용한 자료로 처리하기 위한 것
        - 패턴 인식 : 컴퓨터에서 그림, 음성 등이 정보를 읽어 들여 이미 기억된 정보와 비교/인식하는 방법
        - 전문가 시스템 : 전문 지식을 컴퓨터에 데이터베이스화해 비전문가의 질문에 대한 답을 컴퓨터가 제시하는 시스템

3) 컴퓨터의 분류
- 처리 능력에 따른 분류
    - 슈퍼 컴퓨터(초대형 컴퓨터) : 초당 30~50테라플롭스(TFlops) 속도로 연산
        - 인공위성 제어, 일기예보, 시물레이션 처리 등 특수 분야에 사용됨
    - 메인 프레임(대형 컴퓨터) : 수백 명의 사용자가 동시에 사용할 수 있음
        - 은행, 병원, 정부기관 등
    - 미니 컴퓨터(중형 컴퓨터) : 중규모 시스템
        - 학교/연구소 등의 업무 처리나 과학 기술 계산에 사용
    - 마이크로 컴퓨터(소형 컴퓨터) : 마이크로프로세서(MPU)를 CPU로 사용
        - 네트워크에서 주로 클라이언트 역할
        - 종류
            - 워크스테이션 : 대부분 RISC를 프로세서로 사용, 네트워크에서 서버 역할
                - 고성능 그래픽 처리나 공학용 시물레이션에 주로 사용됨
            - 데스크톱 컴퓨터 : 책상에 놓고 사용하는 일반적인 컴퓨터
            - 휴대용 컴퓨터 : 가볍고 크기가 작아 휴대가 가능한 컴퓨터
- 데이터 취급에 따른 분류
    - 디지털 컴퓨터 : 문자나 숫자가 비연속적인 데이터(디지털형)를 처리하는 일반적 컴퓨터
    - 아날로그 컴퓨터 : 온도, 전류, 속도 등 연속적으로 변화하는 데이터(아날로그형)를 처리하는 특수 목적형
    - 하이브리드 컴퓨터 : 장점 혼합
- 사용 용도에 따른 분류
    - 범용 컴퓨터 : 여러 분야에서 다양한 용도로 사용 -> 디지털 컴퓨터
    - 전용 컴퓨터 : 특수한 목적에만 사용 -> 자동 제어 시스템, 항공기술 등 산업용 제어 분야 -> 아날로그 컴퓨터

핵심요약
- 컴퓨터의 세대별 특징
    - 제1세대 : 진공관
    - 제2세대 : 트랜지스터(TR)
    - 제3세대 : 집적회로(IC)
    - 제4세대 : 고밀도 집적회로(LSI)
    - 제5세대 : 초고밀도 집적회로(VLSI)
- 컴퓨터읩 분류
    - 처리 능력 : 슈퍼 컴퓨터, 메인 프레임, 미니 컴퓨터, 마이크로 컴퓨터
    - 데이터 취급 : 디지털 컴퓨터, 아날로그 컴퓨터, 하이브리드 컴퓨터
    - 사용 용도 : 범용, 전용
