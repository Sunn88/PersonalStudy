* 플립플롭의 기능과 종류를 묻는 문제가 종종 나옴
    - 플립플롭은 1비트를 저장할 수 있음
    - 플립플롭의 종류에는 IS, D, JK, T가 있음

1) 개요
- 순서논리회로 : 외부로부터의 입력과 현재 상태에 따라 출력이 결정되는 회로
    - 기억 기능이 있음
    - 구성 : 플립플롭, 논리 게이트

2) 플립플롭(FF; Flip-Flop)의 특징
- 전원이 공급되고 있는 한, 상태의 변화를 위한 신호가 발생할 때까지 현재의 상태를 그대로 유지하는 논리회로
- 플립플롭 1개가 1Bit를 저장할 수 있음
- 기본적인 플립플롭은 2개의 NAND 또는 NOR 게이트를 이용해 구성함
- 종류 : RS-FF, D-FF, JK-FF, T-FF

3) RS 플립플롭(Reset-Set FF)
- 플립플롭의 기본
- S와 R선의 입력을 조절해 임의의 Bit 값을 그대로 유지시키거나, 무조건 0 또는 1의 값을 기억시키기 위해 사용
- 특성표 : 입력(Q(t)), 출력(Q(t+1))
    S   R   Q(t+1)   상태
    0   0    Q(t)    상태 변화 없음 : 클럭 펄스(CP)가 발생하고 각각 0과 0이 입력되면 플립플롭에 기억되어 있던 원래의 상태값은 변하지 않고 Q(t)값 유지(무)
    0   1     0      Reset          : 0과 1이 입력되면 원래 기억되어 있던 상태값과 상관없이 무조건 0의 상태로 만듦 -> 리셋(항상 0)
    1   0     1      Set            : 1과 0이 입력되면 원래의 상태와 상관없이 무조건 1의 상태로 만듦 -> 세트(항상 1)
    1   1  동작안됨  동작안됨       : S와 R선 모두 1이 입력되면 플립플롭은 동작하지 않음(불가)

4) D 플립플롭
- RS FF의 R선에 인버터(Inverter)를 추가해 S선과 하나로 묶어서 입력선으로 하나만 구성한 플립플롭
- 입력하는 값을 그대로 저장하는 기능 수행
- 특성표
    D   Q(t+1)   상태
    0     0      Reset
    1     1      Set

5) JK 플립플롭
- RS FF에서 S = R = 1일 때 동작되지 않는 결점을 보완한 플립플롭
- RS FF의 입력선 S와 R을 JK FF의 입력선 J와 K로 사용함
- 다른 모든 플립플롭의 기능을 대용할 수 있으므로 응용 범위가 넓고 집적회로화되어 가장 널리 사용됨
- 특성표
    J   K   Q(t+1)   상태
    0   0    Q(t)    상태 변화 없음
    0   1     0      Reset
    1   0     1      Set
    1   1    Q(t)`   반전(보수)

6) T 플립플롭
- JK FF의 두 입력선을 묶어 1개의 입력선으로 구성한 플립플롭
- T = 0인 경우는 변화가 없고, T = 1인 경우 현재의 상태를 토글시킴
    - 원 상태와 보수 상태의 2가지 상태로만 서로 전환됨
- 누를 때마다 On, Off가 교차되는 스위치를 만들고자 할 때 사용
- 특성표
    T   Q(t+1)   상태
    0    Q(t)    상태 변화 없음
    1    Q(t)`   반전
