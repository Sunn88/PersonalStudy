1) 논리 게이트
- 논리회로를 구성하는 기본적인 전자소자
- 0 또는 1을 입력과 출력으로 사용함
- 44p 그림 참조
- AND : 입력신호가 모두 1일때 1 출력 -> Y = AB => D모양
- OR : 입력신호 중 1개만 1이어도 1 출력 -> Y = A + B => 둥그런 D
- NOT : 입력된 정보를 반대로 변환해 출략-> Y = A` => 삼각형 + 점
- BUFFER : 입력된 정보를 그대로 출력 -> Y = A => 삼각형
- NAND : NOT + AND, 즉 AND의 부정 -> Y = (AB)` => D + 점
- NOR : NOT + OR, 즉 OR의 부정 -> Y = (A + B)` => 둥그런 D + 점
- XOR : 입력되는 것이 모두 같으면 0, 한 개라도 다르면 1 출력 -> Y = A`B + AB` => 둥그런 D + 선
- XNOR : NOT + XOR, 즉 XOR의 부정 => 둥그런 D + 선 + 점
