* ★1자 정수의 종류와 전송 선로의 무왜곡 조건을 묻는 문제가 출제됨
    - 1차 정수의 종류, 전송 선로의 무왜곡 조건 숙지하기
* ★광섬유 케이블에 관련된 문제는 개념, 특징, 구성 요소 등 특정 부분에 한정되지 않고 다양하게 출제됨
* 마이크로파의 특징이 출제됨

1) 통신 회선의 개념
- 단말장치에 입력된 데이터 또는 컴퓨터에서 처리된 결과가 실질적으로 전송되는 물리적인 전송 선로
- 전송 선로의 특성
    - 1차 정수 : 저항(R), 정전용량(C), 인덕턴스(L), 누설 컨덕턴스(G)
    - 2차 정수 : 감쇠저수, 위상정수, 전파정수, 특성 임피던스
    - 전송 선로의 무왜곡 조건 : RC = LG

2) 꼬임선(Twisted Pair Wire, 이중 나선)
- 두 개의 절연된 구리선을 서로 고아 하나의 전송 선로를 구성함
- 여러 쌍의 꼬임선들을 하나로 묶어 케이블을 형성함
- 구리선을 꼬는 이유는 두 선 사이의 전기적 간섭을 최소화하기 위해서
- 가격이 저렴하고, 설치가 간편함
- 거리, 대역폭, 데이터 전송률 면에서 제약이 많음
- 다른 전기적 신호의 간섭이나 잡음에 영향을 받기 쉬움
- 종류
    - 비차폐 트위스트 페어(UTP) : 전화 가입자 선 -> LAN선으로 많이 활용함
    - 차폐 트위스트 페어(STP)

3) 동축 케이블(Coaxial Cable)
- 중심 도체를 플라스틱 절연체를 이용해 감싸고, 이를 다시 외부 도체를 이용해 감싸는 형태
- 꼬임선에 비해 외부 간섭과 누화의 영향이 적음
    - 누화 : 전화 통화중 다른 전화의 내용이 함께 들리는 현상
        - 인접한 전송 매체의 전자기적 상호유도 작용에 의해 생기는 잡음 -> 혼선
- 고주파 특성이 양호하며, 광대역 전송에 적합함
- CATV, 근거리 통신망, 장거리 전화 등에 다양하게 사용됨

4) 광섬유 케이블(Optical Fiber Cable)
- 유리를 원료로 제작된 가느달나 광섬유를 여러 가닥 묶어서 케이블의 형태로 만든 것 -> 광케이블
    - 광섬유 : 매우 가늘고 구부릴 수 있는 전송 매체 -> 유리나 플라스틱으로 만듦
- 데이터를 전기 신호가 아닌 빛으로 바꿔 빛의 전반사 원리를 이용해 전송함
- 유선 매체 중 가장 빠른 속도와 높은 주파수 대역폭을 제공함
- 넓은 대역폭을 제공하므로 데이터의 전송률이 높음
- 대용량, 장거리 전송이 가능함
- 가능고 가벼워 취급이 용이함
- 도청하기 어려워 보안성이 뛰어남
- 원료인 유리는 절연성이 좋아 전자 유도의 영향을 받지 않음(무유도성)
    - 전자기적 문제가 최소화되어 안정된 통신 및 누화 방지가 가능함
- 광통신의 3요소
    - 발광기(LD; Laser Diode) : 전광 변환(전기 에너지 -> 빛 에너지), 송신 측 요소
    - 수광기(PD; Hpoto Diode) : 광전 변환(빛 에너지 -> 전기 에너지), 수신 측 요소
    - 광심선(광 케이블) : 중계부 -> 석영, 유리로 구성
- 광섬유 케이블의 구성
    - 코어(Core) : 빛이 전파되는 영역 -> 클래드보다 높은 굴절률을 갖음
    - 클래딩(Cladding) : 코어보다 약간 낮은 굴절률을 가짐 -> 코어의 빛이 외부로 빠져나가지 못하게 하고 코어를 외부로부터 보호함
    - 재킷(Jacket) : 습기, 마모, 파손 등의 위험으로부터 내부를 보호함

5) 라디오파
- 통신 장비의 이동이 빈번하고 통신 회선을 이용하기 어려운 지역 간의 통신에 이용하도록 무선 주파수를 사용하는 방식
    - 주파수 : 단위 시간 내에 신호 파형이 반복되는 횟수
- 주로 장거리 통신 서비스에 이용됨
- 동축 케이블이나 광섬유 케이블의 대용으로 TV나 휴대폰 등의 음성 전송에 이용됨
- 감쇠율이 적어 동축 케이블에 비해 중계기가 훨씬 적게 듬
    - 중계기 : 감쇠 현상 해결
        - 아날로그 전송 : 증폭기를 이용해 신호의 세기를 주파수에 따라 증폭
        - 디지털 전송 : 리피터를 사용해 새로운 신호를 생성(재생)

6) 지상 마이크로파
- 3~40GHz의 주파수 대역을 사용함
- 라디오파와 마찬가지로 장거리 전송에 높은 전송률을 제공함
- 동축 케이블에 비해 중계기가 적게 듦
- 주로 이동 통신이나 TV의 장거리 중계에 이용됨

7) 위성 마이크로파
- 지상에서 쏘아올린 마이크로 주파수를 통신 위성을 통해 변환, 증폭 후 다른 주파수로 지상에 송신하는 방식
- 주파수 대역은 300~3,000MHz인 UHF(Ultra High Frequency)나 3~30GHz인 SHF(Super High Frequency) 사용
- 대역폭이 넓어 고속/대용량 통신이 가능함
- 통신 비용이 저렴함
- 오류율이 적어 고품질의 정보 전송이 가능함
- 전송 비용이 거리에 무관하게 일정함
- 데이터 전송 시 반드시 통신 위성을 거쳐야 하므로, 전송 지연 시간이 김
