* 프린터 등록 정보 창에서 설정할 수 있는 작업 파악, 스풀링 개념 숙지

1) 프린터 설치
- 장치 및 프린터->프린터 추가->프린터 추가 마법사 지시 따라 설치
- 이미 설치한 프린터를 다른 이름으로 다시 설치 가능
- 프린터마다 개별적으로 이름 붙여 설치 가능
- 여러 개의 프린터를 한 대의 컴퓨터에 설치 가능
- 한 개의 프린터를 네트워크로 공유해 여러 컴퓨터에서 사용 가능

2) 프린터 등록 정보 창
- 일반
- 공유
- 포트
- 고급
- 장치 설정

3) 기본 프린터
- 인쇄 명령 수행 시 특정 프린터를 지정하지 않을 경우 자동으로 인쇄 작업이 전달되는 프린터
- 하나만 지정 가능
- 해당 프린터 아이콘에 체크 표시가 되어 있음

4) 스풀 기능
- 중앙처리장치와 같이 처리 속도가 빠른 장치와 프린터와 같이 처리 속도가 느린 장치들 간의 처리 속도 문제 해결하기 위해 사용하는 기능
- 인쇄할 내용을 먼저 하드디스크에 저장
- 백그라운드 작업으로 CPU의 여유 시간에 틈틈이 인쇄
- 프린터가 인쇄중이라도 다른 응용 프로그램을 실행하는 포그라운드 작업 가능

5) 문서 인쇄
- 인쇄 작업이 시작된 문서도 중간에 강제 종료 가능
- 일단 프린터에서 인쇄 작업이 시작된 경우라도 잠시 중지시켰다가 재인쇄 가능
- 인쇄 대기중인 문서를 삭제하거나 대기 순서를 임의 조정 가능
