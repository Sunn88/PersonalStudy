 * 디스크 검사의 기능에 관한 문제
    - 오류 검사 및 치료 하면 디스크 검사
* 디스크 조각 모음의 기능 정리, 수행할 수 없는 매체 기억하기

1) 개요
- [보조프로그램] -> [시스템 도구]
- 디스크 검사, 디스크 조각 모음, 백업, 디스크 정리, 디스크 공간 늘림 등

2) 디스크 검사
- 디스크의 손상된 부분을 점검해 발견된 오류 복구
- 네트워크 드라이브, CD-ROM 드라이브는 수행 불가
- 컴퓨터의 전원이 갑자기 중단되어 다시 부팅될 때는 자동으로 수행됨
- 시스템의 성능 향상을 위해 정기적으로 실행하는 것이 좋음
- 결과창에서 확인 가능한 정보 : 전체 디스크 공간, 불량 섹터, 폴더 수, 숨겨진 파일 용량과 파일 수,
    사용자 파일 수, 사용할 수 있는 공간, 각 할당/전체 할당 단위, 사용할 수 있는 할당 단위 수

3) 디스크 조각 모음
- FAT 구조의 단점인 단편화로 인해 여기 저기 분산되어 저장된 파일들을 연속된 공간으로 최적화
- 디스크의 접근 속도 향싱
- 디스크를 효율적으로 사용하기 위해 파일 정리 -> 용량 증가와는 관계 없음
- Windows가 지원하지 않는 형식으로 압축된 프로그램, CD-ROM 드라이브, 네트워크 드라이브는 수행 불가
- 접근 속도와 안정성이 향상됨
- 걸리는 시간은 볼륨에 있는 파일의 수, 파일 크기, 조각난 양 등에 따라 달라짐
- 수행하는 동안 다른 작업을 할 수 는 있지만, 모든 작업을 중지한 상태에서 해야 효율적으로 조각 모음 가능

4) 백업
- 원본 데이터의 손실에 대비해 중요한 데이터를 외부 저장장치에 하나 더 만들어두는 기능
- 플로피디스크, 자기 테이프, CD-ROM 드라이브, 네트워크 상으 다른 컴퓨터에 가능

5) 디스크 정리
- 디스크의 여유 공간 확보 위해 디스크 드라이브를 검색해 필요 없는 파일을 삭제하는 기능
- 해당 목록 : 임시 인터넷 파일, 오프라인 웹 페이지, 다운로드한 프로그램 파일, 휴지통 파일, 임시 파일
- 기타 옵션 탭을 이용해 사용하지 않는 Windows 구성 요소와 설치된 프로그램 제거 가능
