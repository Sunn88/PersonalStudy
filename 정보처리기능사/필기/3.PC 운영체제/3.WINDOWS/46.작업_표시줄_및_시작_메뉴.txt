* 작업 표시줄의 개념 및 특징에 대한 문제가 다양하게 출제됨

1) 작업 표시줄
- 현재 실행되고 있는 프로그램 단추와 프로그램을 빠르게 실행하기 위해 등록한 고정 프로그램 단추 등이 표시되는 곳
- 기본적으로 바탕 화면 맨 아래쪽에 위치
- 시작 단추, 실행중인 프로그램 단추, 입력 도구 모음, 알림 영역(표시기), '바탕 화면 보기' 단추
- 위치를 변경하거나 크기 조절 가능 -> 화면 1/2까지만
- '작업 표시줄 잠금'이 지정된 상태에서는 크기나 위치 등 변경 불가
- '작업 표시줄 및 시작 메뉴 속성' 대화상자에서는 작업 표시줄 설정 외에 시작 메뉴에 표시할 항목과 전원 단추 동작 등 지정 가능

2) 시작 메뉴
- 작업 표시줄 가장 왼쪽 시작 단추 눌렀을 떄 나타나는 메뉴
- Windoews에 서맃된 프로그램들이 메뉴 형태로 등록되어 있음
- 자주 사용하는 프로그램을 시작 메뉴에 추가해 빠르게 실행할 수 있음
- 바로 가기 키 : Ctrl + Esc
