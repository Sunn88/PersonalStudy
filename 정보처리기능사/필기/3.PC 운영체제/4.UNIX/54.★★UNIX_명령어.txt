* 시스템 및 프로세스 관련 명령어는 대부분의 명령어가 자주 출제됨
    - 각 명령어의 기능을 기억하기
    - 시스템 종료 방법에는 logout, Ctrl_D, exit
* ★특정 기능을 수행하기 위한 명령어를 묻는 문제, UNIX와 DOS 명령 비교 문제
    - pwd, ls 명령어를 중심으로 각각의 기능 정리
    - cp, rm, cal, mv 명령어를 중심으로 각각의 기능 정리
* ls 명령으로 표시되는 파일 종류에 대한 문제가 출제됨
    - 구분자별 파일 유형을 구분할 수 있도록 정리하기


1) 명령어의 기본 형식
- [프롬프트] 명령어 [옵션] [매개변수]
- 프롬프트 : 쉘(Shell)의 종류에 따라 다르게 표시됨($, %, #)
- 명령어 :  반드시 대소문자를 구별해 입력함
- 명령어와 옵션 사이에는 반드시 공백이 있어야 함
- 옵션 지정 시에는 앞에 '-'를 붙임

2) 시스템 및 프로세스 관련 명령어
- kill : 현재 실행중인 프로세스를 삭제(종료) -> 편집 상태에서는 한 줄 전체를 지움
- fork : 새로운 프로세스를 생성(하위 프로세스 호출, 프로세스 복제)
- exec : 새로운 프로세스를 수행
- finger : 현재 시스템에 등록되어 있는 사용자 정보를 조회함
- ps : 현재 작업중인 프로세스의 상태 정표 표시 -> 프로세스의 이름, 명령어 이름, 프로세스 ID 번호, 제어 단말기와 소유주를 포함하는 속성의 목록, 수행된 시간 등
- ping : 특정 시스템과 접속이 안 될 경우 네트워크 상의 문제 진단
- login : UNIX 작업을 위해 사용자 ID와 비밀번호를 입력받아 사용자를 확인함
- logout : UNIX 시스템에 대한 접속을 종료함(Ctrl + D, exit)
- passwd : 로그인할 떄 필요한 비밀번호를 설정 도는 변경함
- who : 로그인한 사용자에 관한 정보 표시 -> 단말기명, 로그인명, 로그인 일시, 사용중인 단말기 번호 등
- mount : 파일 시스템을 마운팅/마운팅 해제함
    - 파일 시스템 마운팅 : 새로운 파일 시스템을 기존 파일 시스템의 서브 디렉터리에 연결하는 것
- dump : 주기억장치의 내용을 화면이나 프린터, 디스크 등에 출력함

3) 디렉터리 관련 명령어
    UNIX명령어  DOS명령어   기능
    pwd                     현재 작업중인 디렉터리 경로를 화면에 출력함 
    ls          dir         현재 작업중인 디렉터리의 모든 파일을 표시함
    mkdir       md          디렉터리 생성
    rmdir       rd          디렉터리 제거
    cd          cd          현재 작업중인 디렉터리에서 다른 디렉터리로 이동함
    .           .           현재 디렉터리
    ..          ..          상위 디렉터리
    /           \           루트 디렉터리
- ls 명령으로 표시되는 파일의 유형
    - - : 일반 파일(이진 파일)
    - b : 블록형 특수 파일1
    - c : 문자형 특수 파일
    - d : 디렉터리
    - l : 링크 파일

4) 파일 관련 명령어
    UNIX명령어  DOS명령어       기능
    creat                       파일 생성
    open                        파일을 사용할 수 있는 상태로 준비시킴
    close                       파일 닫기
    cp          copy, xcopy     파일 복사
    rm          del             파일 삭제
    cat         type            파일의 내용을 화면에 표시함
    chmod       attrib          파일의 보호 모드를 설정해 파일의 사용 허가 지정
    chown                       소유자 변경
    find                        파일 찾기
    mv                          파일을 이동시키거나 이름 변경
    wc                          파일 내의 라인 수, 단어 수, 문자 수 등 표시
    cmp                         두 파일을 비교해 차이가 나는 바이트 위치와 행 번호 표시
    comm                        두 파일을 비교해 한족에만 있거나 양쪽에 모두 있는 라인 찾아 표시
    diff        fc              두 파일을 비교해 차이점 표시
    paste                       하나의 파일 끝에 다른 파일의 내용 추가
