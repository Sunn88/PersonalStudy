* ★데이터 신호 속도 계산 문제가 자주 출제됨
    - 데이터 신호 속도의 계산식과 변조시 상태 변화 수 기억하기
* 전송로의 통신 용량을 늘리기 위한 방법을 묻는 문제가 출제됨
    - 통신 용량을 늘리려면 신호의 통로인 대역폭을 넓히고, 신호를 방해하는 잡음은 줄여야 함
    - 신호가 세면 더 많은 양의 신호가 통과함

1) 통신 속도
- 변조 속도 : 1초 동안 몇 개의 신호 변화가 있었는가를 나타내는 것 -> 단위는 baud
    - 1개의 신호가 변조되는 시간을 T초라고 할 때 변조 속도 baud = 1/T
- 신호 속도 : 1초 동안 전송 가능한 비트의 수 -> 단위는 bps(bit/sec)
    - 데이터 신호 속도(Bps) = 변조 속도(Baud) * 변조 시 상태 변화 수
        - 변조 시 상태 변화 수
            - 모노비트(Monobit) : 1비트
            - 디비트(Dibit) : 2비트
            - 트리비트(Tribit) : 3비트
            - 쿼드비트(Quadbit) : 4비트
    - 변조 속도(Baud) = 데이터 신호 속도(Bps) / 변조 시 상태 변화 수
    - 변조 시 상태 변화수가 1Bit인 경우 Bps = Baud
- 전송 속도 : 단위 시간에 전송되는 데이터의 양(문자, 블록, 비트, 단어 수 등)
    - 단위 : 문자/초(분, 시간), 블록/초(분, 시간), 단어/초 등과 같은 형식

2) 통신 용량
- 단위 시간 동안 전송 회선이 최대로 전송할 수 있는 통신 정보량
- 샤논(Shannon)의 정의 : 전송로의 대역폭과 신호, 잡음을 고려하여
    C = W * log(1 + S/N)[bps] => C(통신 용량), W(대역폭), S(신호 전력), N(잡음 전력)
- 전송로의 통신 용량을 늘리기 위한 방법
    - 주파수 대역폭을 늘림
    - 신호 세력을 높임
        - 신호 세력 : 신호 세기의 강하거나 약한 정도 -> 데시벨(dB)
    - 잡음 세력을 줄임
