* 드래그 앤 드롭을 중심으로 사용 방법 알아두기

1) 마우스 사용법
- 클릭 : 마우스 왼쪽 단추 한 번 누르는 동작 -> 아이콘이나 메뉴, 창 선택시 사용
- 더블클릭 : 마우스 왼쪽 단추 빠르게 두 번 누르는 동작 -> 창이나 폴더를 열거나 응용 프로그램 실행시 사용
- 드래그 앤 드롭 : 마우스 왼쪽 단추를 누른 채 끌다가 놓는 동작 -> 파일, 폴더를 이동/복사, 창 크기 조절, 바로 가기 아이콘 생성시 사용
- 오른쪽 단추 클릭 : 바로 가기 메뉴를 나타낼 떄 사용

2) 바로 가기 키(단축키)
- 키보드의 키를 조합해 명령어 대신 사용하는 기능 -> 단축기 또는 핫키(Hot Key)
- 기능키 : F1, F2, F3, F5 -> 도움말 보기, 폴더/파일 이름 바꾸기, 찾기, 최신 정보로 고침
- Alt +
    - 좌우화살표 : 현재 실행중인 화면의 다음 화면이나 이전 화면으로 이동함
    - Esc : 현재 실행중인 프로그램들을 순서대로 전환함
    - Tab : 현재 실행중인 프로그램을의 목록을 화면 중앙에 나타냄
    - Enter : 선택된 항목의 등록 정보를 나타냄
        - 도스 전체 화면과 도스 창 형태를 번갈아가며 표시함
    - Spacebar : 현재 열려 있는 창의 제어 상자(창 조절 메뉴) 표시
    - F4 : 실행중인 창이나 응용 프로그램 종료 -> 실행중인 프로그램이 없으면 시스템 종료
    - Print Screen : 현재 작업중인 활성 창을 클립보드로 복사함
- Print Scree : 화면 전체를 클립보드로 복사함
- Ctrl +
    - A : 폴더 및 파일 모두 선택
    - Esc : 시작 메뉴 표시
    - Shift + Esc : 작업 관리자 대화상자를 실행해 문제가 있는 프로그램 강제 종료
- Shift +
    - Delete : 폴더나 파일을 휴지통을 거치지 않고 바로 삭제함
    - F10 : 바로 가기 메뉴 표시
    - CD 삽입 : CD의 자동 실행 기능이 작동하지 않음

3) 윈도우키를 이용한 바로 가기 키
- 윈도우키 : 시작 메뉴 표시
- 윈도우키 + R : 실행 창 표시
- 윈도우키 + P : 화면 우측에 프로젝트 바로 가기 표시
- 윈도우키 + E : 파일 탐색기 실행
- 윈도우키 + M, + Shift + M : 열려 있는 모든 창을 최소화/이전 크기로 표시함
