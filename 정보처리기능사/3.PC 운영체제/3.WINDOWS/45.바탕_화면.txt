* 바로 가기 아이콘의 전체적인 특징을 묻는 문제가 자주 출제됨

1) 바탕 화면의 구성
- Windows 설치 시 기본적으로 표시되는 아이콘과 작업 표시줄로 구성됨

2) 바탕 화면의 바로 가기 메뉴
- 바탕 화면 작업 시 자주 사용하는 명령을 메뉴로 구성한 것
- 바탕 화면에서 마우스 오른쪽 단추를 클릭시 표시됨
- 바로 가기 메뉴 : 보기, 정렬 기준, 새로 고침, 새로 만들기, 화면 해상도, 가젯, 개인 설정 등
- 바탕 화면에 있는 아이콘의 표시 유무 지정 가능

3) 바탕 화면의 기본 아이콘
- 내 컴퓨터 또는 컴퓨터
    - 컴퓨터에 설치된 디스크 드라이브, 폴더, 파일 등을 표시
    - 파일과 폴더 그리고 컴퓨터에 설치된 하드웨어 관리 가능
- 휴지통 : 삭제된 파일이나 폴더가 임시로 보관되는 곳
- 내 문서 또는 개인 폴더
    - 아이콘 이름은 로그인한 사용자 계정의 이름으로 표시됨
    - 해당 계정으로 로그인해 작업한 내용이 기본적으로 저장되는 곳
    - 실제 폴더 위치는 C:\사용자\사용자계정
- 네트워크 환경 또는 네트워크
    - 네트워크가 구축되어 있을 때만 표시되는 아이콘
    - 네트워크에 연결된 자원을 확인하고 공유할 수 있음
- 제어판 : 컴퓨터를 구성하는 프로그램과 하드웨어에 대한 설정 사항 등을 변경하는 여러 가지 항목이 표시되는 곳

4) 바로 가기(단축) 아이콘
- 개요 및 특징
    - 자주 사용하는 문서나 프로그램을 빠르게 실행시키기 위한 아이콘 -> 실제 실행 파일과 연결해 놓은 것
    - 폴더나 파일, 디스크 드라이브, 다른 컴퓨터, 프린터 등 모든 개체에 대해 작성할 수 있음
    - 왼쪽 아랫부분에 화살표 표시가 있어 일반 아이콘과 구별됨
    - 확장자는 LNK, 컴퓨터에 여러 개 존재할 수 있음
    - 사용자가 임의로 생성하거나 삭제할 수 있음
    - 삭제하더라도 원본 파일은 삭제되지 않음
- 만들기
    - 메뉴 이용
    - 바로 가기 메뉴 이용
    - 오른쪽 버튼으로 끌기
    - Ctrl+Shift+드래그
