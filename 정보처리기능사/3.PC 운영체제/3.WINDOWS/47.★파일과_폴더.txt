* 폴더의 개념과 파일명으로 사용할 수 없는 문자 알아두기
* Windows 탐색기의 기능을 묻는 문제가 출제됨
    - 어떤 작업을 할 수 있는지 알아두기
* Windows 탐색기에서의 작업을 컴퓨터를 통해서도 수행할 수 있음

1) 파일과 폴더의 특징
- 파일 : 디스크에 저장되는 기본 단위
- 폴더 : 파일을 모다 관리하기 위한 장소 -> 도스에서 디렉터리와 유사 개념
- 파일은 파일명과 확장자로 구성됨 -> 마침표를 이용해 구분
- 파일과 폴더는 작성, 이름 변경, 삭제가 가능 -> 하위 폴더나 파일이 포함된 폴더도 삭제 가능
- 하나의 폴더 내에는 동일한 이름의 파일이나 폴더가 존재할 수 없음
- 파일과 폴더의 이름은 255자 이내, 공백 포함 가능
- * / ? \ : < > " | 등은 파일과 폴더의 이름으로 사용 불가

2) Windows 탐색기
- 컴퓨터에 설치된 디스크 드라이브, 응용 프로그램 파일 및 폴더 등을 관리할 수 있는 곳
- 파일이나 폴더, 디스크 드라이브에 관련된 모든 작업 수행 가능
- 컴퓨터와 파일과 폴더를 계층(트리) 구조로 표시함
- 크게 탐생 창과 파일 영역(폴더 창) 두 부분으로 나눠짐
    - 탐색 창 : 컴퓨터에 존재하는 모든 폴더가 표시됨
    - 파일 영역 : 탐색창에서 선택한 폴더의 내용(하위 폴더, 파일)이 표시됨

3) 파일/폴더 이름 바꾸기
- [파일] 메뉴 이용
- 바로 가기 키 이용
- 바로 가기 메뉴 이용
- 마우스 사용
- 휴지통은 이름 변겨 ㅇ불가

4) 파일/폴더 선택
- 하나의 항목 선택 : 왼쪽 클릭
- 연속적인 항목 선택 : 드래그, Shift누른 상태에서 마지막 클릭
- 비연속적인 항목 선택 : Ctrl 누린 상태에서 차례로 클릭
- 전체 항목 선택 : Ctrl+A
- 선택 항목 반전 : [편집]->[선택 항목 반전]

5) 파일/폴더 찾기
- 파일과 폴더의 이름을 입력해 찾을 수 있음
- 만능 문자(와일드 카드, *, ?) 이용 가능
- 특정 문자열을 포함한 파일 찾을 수 있음
- 하위 폴더까지 검색할지 여부 지정 가능
