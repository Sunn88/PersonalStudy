* 파일/폴더를 복사하거나 이동할 때 사용되는 키 구분하기
    - 복사는 Ctrl, 이동은 Shift
* ★클립보드에 대한 문제가 자주 출제됨
    - 클립보드의 개념, 클립보드로 내용이 저장되게 하는 단축기 기억하기
* ★휴지통 사용 방법에 대한 문제가 자주 출제됨
    - 휴지통의 기능, 휴지통에 버리지 않고 바로 삭제할 때 사용하는 키(Shift) 중심으로 내용 정리

1) 파일/폴더 복사
- [편집]메뉴 이용 : 복사
- 바로 가기 키 이용 : Ctrl+C + Ctrl+V
- 도구 모음 이용
- 키보드와 마우스 사용

2) 파일/폴더 이동
- [편집]메뉴 이용 : 잘라내기
- 바로 가기 키 이용 : Ctrl+X + Ctrl+V
- 도구 모음 이용
- 키보드와 마우스 사용

3) 클립보드
- 데이터를 일시적으로 보관해 두는 임시 저장공간 -> 일종의 버퍼 역할
- 서로 다른 응용 프로그램 간에 데이터를 쉽게 전달할 수 있음
- 클립보드의 내용은 여러 번 사용이 가능하지만, 가장 최근에 저장된 것 하나만 기억함
- 시스템을 재시작하면 클립보드에 저장된 데이터는 지워짐
- 복사/잘라내기, 붙여넣기, 활성창 복사(Alt+PrintScreen), 전체 창 복사(PrintScreen)시 사용됨
- 클립보드에 저장된 데이터는 클립보드 표시기를 통해 볼 수 있음
- 클립보드 표시기를 이용해 클립보드의 내용을 삭제하거나 다른 이름으로 저장해 사용할 수 있음

4) 휴지통 사용하기
- 삭제된 파일이나 폴더가 임시 보관되는 장소
- 크기는 기본적으로 드라이브 용량으 10%로 설정됨 -> 최소 0%부터 최대 100%까지 임의 설정 가능
- 복원 가능 -> 복원 전에는 사용 불가
- 휴지통 비우기를 수행하고 나면 복원할 수 없음
- 휴지통의 크기를 드라이브마다 다르게 도는 모두 동일한 크기로 설정 가능
- 휴지통을 거치지 않고 바로 삭제하도록 설정 가능
- 삭제될 때마다 확인 메시지가 표시되도록 설정 가능
