* DOS에서 디렉터리를 묻는 문제가 출제됨
* ★부팅의 개념, 부팅 시 반드시 필요한 파일, 부팅 과정 시 사용되는 파일의 순서, 웹 부팅과 콜드 부팅의 개념 등
* 숨김 속성이 있는 시스템 파일, 실행 파일의 확장자 등에 대한 문제
* 기능키 중 F3의 기능 알아두기

1) 개요
- DOS(Disk Operating System) : 개인용 컴퓨터에서 디스크와 파일 관리를 위한 운영체제
- 단일 처리 방식
- CUI 환경의 대표적인 운영체제
    - CUI(Character User Interface) : 사용자가 문자 환경을 통해 컴퓨터와 정보를 교환하는 작업 환경
- 디렉터리(Directory) : 파일의 저장, 보관, 관리 등 효율적인 파일 사용을 위해 디스크에 존재하는 파일에 대한 여러 정보를 가지고 있는 특수한 형태의 파일

2) 부팅 과정
- 부팅(Booting, Bootstrap) : 컴퓨터에 전원을 넣은 순간부터 DOS(시스템 프로그램)를 디스크로부터 주기억장치로 읽어 내어 컴퓨터를 이용할 수 있는 상태로 만들어 주는 과정
- 부팅에 반드시 필요한 시스템 파일 : MS-DOX.SYS, IO.SYS, COMMAND.COM
- 부팅 중 F8을 눌러 'CONFIG.SYS'파일과 'AUTOEXEC.BAT' 파일의 실행 여부 선택 가증
- 부팅 순서
    컴퓨터 전원 공급
    ->ROM BIOS를 이용해 주기억장치와 각 장치의 이상 유무 검사
    ->IO.SYS 파일을 주기억장치에 적재
    ->MSDOS.SYS 파일을 주기억장치에 적재
    ->CONFIG.SYS 파일을 읽어 시스템 환경 설정
    ->COMMAND.COM 파일 실행
    ->AUTOEXEC.BAT 파일 실행
    ->명령 프롬프트 표시
- 프롬프트(Prompt) : 사용자의 명령을 받아들일 수 있는 상태
    - $, %, =와 같은 형태로 변경할 수 있음
- 종류
    - 웜 부팅(Warm Booting)
        - ctrl+Alt+Delete를 눌러 컴퓨터를 재부팅하는 소프트웨어적 부팅
    - 콜드 부팅(Cold Booting)
        - 컴퓨터 본체의 전원 스위치를 눌러 켜거나 본체의 리셋 단추를 눌러 재부팅하는 것

3) 파일
- 파일명 작성 규칙
    - 영문 8자 이내, 확장자 3자로 구성됨 -> 확장자로 파일 구분
    - 공백, \ / * ? ; > < | 등 사용 불가
    - 시스템 예약 파일 AUX, CON, PRN, NUL 등 사용 불가
        - AUX : COM1~COM4와 같은 보조 입출력 포트
        - CON : 표준 입출력장치 -> 키보드, 모니터 등
        - PRN : 프린터(LPT1)
        - NUL : 가상장치 파일
- 종류
    - 시스템 파일
        - 시스템을 운용 및 관리하는 파일
        - MSDOS.SYS : 파일의 입출력 및 시스템 호출에 필요한 장치 관리(숨김 속성 파일)
            - 숨김 속성 : 시스템 파일 등과 같이 중요한 파일을 보호하기 위해 파일의 존재를 확인할 수 없도록 한 속성
        - IO.SYS : MSDOS.SYS의 요구에 의해 실제로 입출력을 수행함(숨김 속성 파일)
        - COMMAND.COM  : 명령어 해석기 -> 부팅 시 내부 명령어를 주기억장치에 적재시키고, 사용자가 입력한 명령어를 처리함        
    - 실행 파일
        - EXE, COM, BAT 등의 확장자를 가진 파일 -> 파일명을 입력 후 Enter를 누르면 실행됨
    - 배치 파일
        - 확장자가 BAT인 파일 -> 명령어가 모여 있는 파일
    - 백업 파일
        - 데이터 훼손을 대비해 동일한 파일을 하나 더 만든 파일
    - 텍스트 파일
        - 아스키 형태의 파일
- 템플릿트(Template) 기능
    - 사용자가 바로 전에 사용한 명령어를 저장하고 있다가 특정 키가 입력되면 그 키의 기능에 따라 문자를 프롬프트에 표시해주는 것
    - F1 : 바로 전에 입력한 명령을 한 문자씩 표시
    - F2 : 바로 전에 입력한 명령 중 지정한 문자 앞까지 표시
    - F3 : 바로 전에 입력한 명령을 모두 표시
    - F4 : 바로 전에 입력한 명령 중 지정한 문자 앞까지 삭제하고 뒷부분 표시
    - F6 : 'COPY CON' 명령으로 텍스트 파일 작성 후 저장시킴
        - COPY CON : 간단한 문서 파일을 작성하고 확인할 수 있는 내부 명령어
