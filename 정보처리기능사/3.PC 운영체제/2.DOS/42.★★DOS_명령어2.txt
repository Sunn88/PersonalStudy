* 각 명령어의 영문 의미를 통해 기능 정리하기
* 파일 명령어의 DIR과 ATTRIB 명령은 세부 옵션까지 알아두기

1) 디스크 관련 명령어
- FORMAT
    - 디스크에 데이터가 저장될 수 있도록 트랙과 섹터를 만드는 초기화 작업 수행
    - 옵션
        - /S : 포맷한 후 시스템 파일을 복사해 부팅 가능한 디스크로 만듦
        - /Q : 이미 사용하던 디스크의 빠른 포맷
        - /V[:이름] : 포맷한 후 디스크에 볼륨명 부여
        - /F : 용량을 지정해 포맷
        - /4 : 고밀도 드라이브(1.2M)에서 360KB 형식으로 포맷
    - 형식 : FORMAT [드라이브:] [/옵션]
- DISKCOPY
    - 원본 디스크에 있는 내용을 모두 다른 디스크로 복사
    - 같은 용량의 디스크에서만 사용 가능 -> 하드디스크에서는 사용 불가
    - 형식 : DISKCOPY [원본 드라이브:] [사본 드라이브:]
- CHKDSK
    - 디스크의 상태를 점검하고 결과 표시
    - 형식 : CHKDSK [드라이브:] [/옵션]
- FDISK
    - 하드디스크를 논리적으로 여러 개의 디스크로 나눠(파티션 설정)
    - 각 볼륨이 서로 다른 드라이브 문자를 가진 별개의 드라이브로 동작하도록 설정
- DEFRAG
    - 단편화되어 있는 파일의 저장 상태를 최적화해 디스크의 작동 효율을 높임
- SYS
    - 도스의 시스템 파일을 디스크에 복사하는 것 -> 해당 디스크로 부팅 가능
- DISKCOMP
    - 동일한 디스켓인지 서로 비교
    - DISKCOPY 명령으로 복사된 디스크가 정상적으로 복사되엇는지 판별할 떄 유용하게 사용됨
- SCANDISK
    - 디스크 검사 유틸리티 -> 디스크의 문제점 진단, 수정
    - 하드디스크의 표면을 검사해 표면에 존재하는 미미한 오류 수정 가능

2) 디렉터리 관련 명령어
- MD(Make Directory)
    - 새로운 디렉터리를 만드는 명령어
    - 같은 디렉터리에 동일한 이름의 디렉터리를 생성할 수 없음
    - 형식 : MD [드라이브:] [경로] 디렉터리명
- RD(Remove Directory)
    - 디렉터리 삭제
    - 삭제하려는 디렉터리 안에 다른 디렉터리나 파일이 없어야 삭제 가능
    - 형식 : RD [드라이브:] [경로] 디렉터리명
- CD(Change Directory)
    - 다른 디렉터리로 이동
    - 형식 : CD [드라이브:] [경로] 디렉터리명
- DELTREE
    - 디렉터리 안에 있는 파일과 디렉터리까지 모두 삭제함
    - DELTREE [드라이브:] [경로] 디렉터리명
- XCOPY
    - 특정한 디렉터리 내의 모든 파일 및 하위 디렉터리까지 복사 가능
    - 숨김 파일과 시스템 파일은 복사되지 않음
    - 옵션
        - /P : 각 파일을 복사할 때마다 취소할 수 있도록 확인 메시지를 표시함
        - /A : 파일에 저장 속성이 설정된 파일만 복사함
        - /S : 비어 있지 않은 디렉터리와 하위 디렉터리를 모두 복사함
        - /D : 특정 날짜 이후 변경된 파일만 복사
    - 형식 : XCOPY [원본 드라이브:] [경로] 파일명 [대상 드라이브:] [경로] 파일명 [/옵션]
- PATH
    - 실행 파일을 찾는 경로를 설정하거나 보여주는 데 사용됨
    - 서로 다른 드라이브에 있는 파일도 PATH에 지정되면 검색 가능
    - 찾고자 하는 파일이 현재의 디렉터리에 없을 때에만 PATH에서 지정한 경로를 검색함
    - 경로와 경로 사이는 ;로 구분
    - 경로를 해제할 떄는 PATH; 사용
    - 형식 : PATH [경로] ; [경로]

3) 파일 명령어
- DIR(DIRectory)
    - 디스크 내에 수록된 파일 및 디렉터리에 대한 정보 표시
    - 옵션
        - /P : 목록을 한 화면 단위로 표시
        - /W : 한 줄에 5개씩 목록을 출력해주는 것 -> 화면에 가장 많은 파일을 표현할 수 있는 방식
        - /O : 지정한 정렬 방식으로 파일 목록 표시 -> 역순 표시는 '/O-'
            - D(Date) : 날짜/시간 순으로 정렬(빠른 순)
            - E(Extension) : 확장자 순
            - N(Name) : 파일 이름 순
            - S(Size) : 파일 크기(Byte) 순(작은 것부터)
        - /S : 하위 디렉터리의 정보가지 표시
        - /A : 기록 속성이 설정된 목록 표시
        - /H : 숨겨진 파일 목록 표시
    - 형식 DIR [드라이브:] [경로] 파일명 [/옵션]
- COPY
    - 파일을 지정한 곳에 복사하거나 여러 개의 파일을 결합함
    - 형식 : COPY [원본 드라이브:] 파일명 [복사할 드라이브:] 파일명
- DEL=ERASE
    - 파일 삭제
    - 형식 : DEL [드라이브:] [경로] 파일명 [/옵션]
- UNDELETE
    - DEL이나 ERASE를 사용해 삭제한 파일 복원
    - 형식 : UNDELETE [드라이브:] [경로] 파일명
- TYPE
    - 아스키 코드로 작성된 파일, CONFIG.SYS 파일, 배치 파일, COPY CON 명령으로 작성한 파일 등의 내용을 화면에 출력
    - 형식 TYPE [드라이브:] [경로] 파일명
- REN
    - 파일의 이름 변경 -> 동일한 디렉터리 내에서만 가능
    - 형식 : REN [드라이브:] [경로] 파일명 새로운파일명
- ATTRIB
    - 파일의 속성 표시, 해제, 지정
    - 옵션
        - + : 속성 지정
        - - : 속성 해제
        - R : 읽기 전용 속성
        - A : 저장/백업 속성
        - S : 시스템 파일 속성
        - H : 숨김 파일 속성
    - 형식 : ATTRIB [+속성/-속성] [드라이브:] [경로] 파일명 [/옵션]
- FC
    - 두 개의 파일을 비교해 그 차이를 나타냄
    - 파일 복사 후 정확히 복사되었는지 확인
