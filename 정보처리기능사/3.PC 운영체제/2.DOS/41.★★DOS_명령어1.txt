* 내부 명령어와 외부 명령어의 개념을 묻는 문제가 자주 나옴
    - 내부 명령어와 외부 명령어의 개념과 종류를 구분해 파악하기
* 명령어들의 종류를 묻는 문제가 자주 나옴
* 시스템의 날자, 버전 확인, 화면 지움 등에 사용하는 명령어를 묻는 문제가 자주 나옴
* 필터 명령어의 종류와 기능을 구분하는 문제가 출제됨

1) 내부 명령어
- 도스가 부팅될 떄 COMMAND.COM이 실행됨과 동시에 주기억장치에 상주하는 명령어
- 주기억장치에 상주하므로 언제든지 실행이 가능함
- 비교적 자주 사용되며 실행 과정이 간단함
- COMMAND.COM이 관리하는 명령어로 COMMAND.COM 파일이 주기억장치에 올려짐으로써 사용할 수 있음
- 내부 명령어는 경로(Path)와 관계없이 어떤 디렉터리에서도 실행 가능
    - 경로 : 디렉터리 간의 관계를 표시해주는 것 -> 각 디렉터리는 '\'로 구분
- 종류 : CLS, DIR, VER, COPY, DATE, TIME, MD, CD, RD, PROMPT, VOL, TYPE, DEL 등

2) 외부 명령어
- 보조기억장치에 저장되어 있다가 사용자가 명령을 입력하면 주기억장치에 적재시킨 후 실행하는 명령어
- 종류 : ATTRIB, FORMAT, CHKDSK, FDISK, LABEL, SCANDISK, DISKCOPY, XCOPY, SORT, SYS 등

3) 기본 명령어
- CLS : 현재 화면에 표시된 내용을 지움
- DATE : 현재 시스템의 날짜를 확인하거나 변경함
- TIME : 현재 시스템의 시간을 확인하거나 변경함
- VER : 현재 사용중인 DOS의 버전을 표시함
- PROMPT : DOS의 프롬프트를 여러 가지 형태로 변경
- VOL : 드라이브의 볼륨명과 일련번호 표시

4) 필터 명령어
- DOS에서 명령을 실행할 때 조건을 부여해 수행할 수 있는 외부 명령어
- 종류
    - MORE : 내용을 한 화면씩 출력함
    - FIND : 하나 또는 여러 개의 파일에서 특정한 문자열을 검색함
    - SORT : 내용을 정렬해 화면이나 파일로 출력함
