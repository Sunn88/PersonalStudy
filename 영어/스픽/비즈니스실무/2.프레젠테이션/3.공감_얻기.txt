1. 실행하다
By implementing this plan, we could see as much as a 20% increase in productivity.
이 계획을 실행함으로써, 생산력에 있어서 무려 20%의 증가를 기대할 수 있습니다.
* implement : 실행하다, 이행하다, 적용시키다
* implement strategy : 전략을 실행하다
* implement policy : 정책을 도입하다
* implement system : 시스템을 구현하다
* implement plan : 계획을 이행하다
* significant cut : 엄청난 감소


2. 일맥상통 하다
Outsourcing production is in line with the new company strategy.
제작을 아웃 소싱하는 것은 새로운 회사 전략과도 일치합니다, 일맥상통합니다.
* in line with : 일치하다
Hiring new staff is in line with the company`s priotiries.
새로운 직원 고용은 회사의 우선순위와도 일치해요.


3. 자그마치, 무려
* as much as : 무려
