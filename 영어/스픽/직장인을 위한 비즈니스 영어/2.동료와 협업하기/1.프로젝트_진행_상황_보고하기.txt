I dropped the ball.
진짜 실수한거죠.

1. We`ve run into some delays.
우리가 지연을 좀 겪고 있습니다.
* run into : 상황이 누구의 잘못도 아니라는 인상을 줌 -> 자연스러운 상황임을 강조

We`ve run into some hurdles.
장애물에 좀 부딪혔습니다.

We`ve run into some roadblocks.
장벽에 좀 부딪혔습니다.
* roadblock : 큰 장애물, 장벽; 프로젝트의 진행을 맏는 큰 문제; 완전한 중단
* run into : 부딪히다; 누군가 또는 무언가를 맞닥뜨리다
* hiccup : 딸꾹질 같은 일시적 지장; 약간의 문제 또는 사고
* delay : 지연; 예정보다 늦어지는 문제
* hurdle : 장애물; 국복해야 하지만 완전한 중단된 것은 아닌 장애물

To avoid any unexpected difficulties, let`s delegate tasks.
예상치 못한 어려움을 피하기 위해 업무를 위임합시다.
* delegate tasks : 업무를 위임하다; 모든 사람에게 명확하고 균등하게 업무를 배정하다
To avoid any roadblocks, let`s plan ahead next time.
장벽을 피하기 위해 다음번에는 미리 계획을 세웁시다.
* plan ahead : 미리 계획을 세우다; 향후의 문제와 결과를 염두에 두고 계획을 세우다
To avoid any unexpected difficulties, let`s monitor our expenses.
예상치 못한 어려움을 피하기 위해 지출을 모니터링합시다.
* monitor expenses : 지출을 모니터링하다; 지출을 면밀히 기록하고 검토하다


2. Can you have it done by Friday?
금요일까지 그것을 완료하실 수 있나요?
* have something done : 누군가를 위해 어떤 일을 완료하다

Can you have the contract signed by tomorrow?
내일까지 계약서 서명을 완료하실 수 있나요?


3. I`ll have the report finished by the end of the week.
이번 주 말까지 보고서를 완료하도록 하겠습니다.
* by the end of the week(EOW) : 금요일까지, 주말까지

I`ll have the doc prepared by the end of the day.
오늘까지 문서를 준비하도록 하겠습니다.
* by the end of the day(EOD) : 업무 시간 마감까지
