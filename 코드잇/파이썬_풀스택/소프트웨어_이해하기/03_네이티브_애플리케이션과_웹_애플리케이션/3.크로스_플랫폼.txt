크로스 플랫폼 : 여러 OS, CPU 등에서 동일하게 실행할 수 있는 환경
- 하나의 프로그램만으로 여러 플랫폼에서 실행할 수 있는 것
- 플랫폼 위에 소프트웨어로 만든 또 하나의 컴퓨터를 실행시켜 다양한 플랫폼을 하나의 컴퓨터처럼 만듦
    - 가상 머신 : 가상의 컴퓨터에 맞춰서 프로그램 개발
        - Java의 JVM(Java Virtual Machine)
        - 이식성과 범용성 상승
    - 웹 : html, JavaScript 등으로 작성 후 웹브라우저가 실행
        - 반응형 웹 디자인 : 더 간편하게 크로스 플랫폼 생성
        - 웹 표준 : 국제 표준회 기구(ISO)에 등록됨
        - 단점 : 브라우저 호환성 문제
            - 브라우저들이 웹 호환성이 100% 지켜지지 않는 이유
                - 사용자들의 수요에 따른 잦은 업데이트
                - 웹 표준 버전과 웹 브라우저 버전이 다름
                - 일부러 웹 표준과 다르게 만듦
            - 얼마나 웹 표준을 잘 지원하는가가 중요
