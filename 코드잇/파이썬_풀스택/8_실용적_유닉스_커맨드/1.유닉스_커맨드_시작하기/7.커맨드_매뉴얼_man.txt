커맨드의 매뉴얼 찾기 : man + argument

매뉴얼 보는 방법
- name : 커맨드 이름, 커맨드의 역할 설명
- synopsis : 커맨드 문법, 사용 가능한 옵션
    - 대괄호 : 필수가 아닌 옵션
    - -가 붙은 것들은 옵션, 아닌 것들은 argument
    - 옵션에도 argument가 붙을 수 있음
- description
    - 일반적인 설명
    - 각 옵션에 대한 설명
    - 유의사항
- 비슷한 커맨드
- 참고사항
- q : 매뉴얼 나가기

cal -B 3 -A 3
cal -j -B 3 -A 3
