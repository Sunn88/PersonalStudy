vim 실행 : vim 커맨드
    vim
- 파일 경로를 아규먼트로 주면 해당하는 파일을 열거나, 존재하지 않으면 새로 만들어줌
    vim path/to/file

vim의 4가지 사용 모드
- 일반 모드 : 커서 이동, 텍스트 붙여넣기, 작업 취소
- 입력 모드 : 텍스트 입력
- 비주얼 모드 : 텍스트 블록 지정, 텍스트 복사
- 명령 모드 : 내용 저장, vim 종료

                i->     입력모드
                <-esc
    일반모드    v,V->   비주얼모드
                <-esc
                :->     명령모드
                <-esc

단축키 정리
- 텍스트 입력: 입력 모드(i) → 텍스트 입력
- 텍스트 한 줄 복사: 일반 모드 → 복사하고 싶은 줄에 커서 위치 → yy
- 텍스트 한 줄 잘라내기: 일반 모드 → 잘라내고 싶은 줄에 커서 위치 → dd
- 특정 영역 복사: 비주얼 모드(V는 줄 단위, v는 글자 단위) → 복사하고 싶은 영역 커서로 설정 → y
- 특정 영역 잘라내기: 비주얼 모드(V는 줄 단위, v는 글자 단위) → 잘라내고 싶은 영역 커서로 설정 → d
- 텍스트 붙여넣기: 일반 모드 → 붙여넣고 싶은 위치에 커서 위치 → p
- 파일 저장: 명령 모드(:) → w + enter
- 파일 저장 + vim 종료: 명령 모드(:) → wq + enter
- vim 종료 (내용 저장되지 않음): 명령 모드(:) → q! + enter
- 파일 내용 출력
    - cat : 내용 전체를 쭉 출력
    - less : 내용을 페이지 단위로 나눠서 출력
- head -n 15 dont_stop_me_now.txt : 현재 디렉토리에 있는 dont_stop_me_now.txt 파일의 처음 15줄만 확인
    - 파일의 일부 내용만 출력하는 head, tail 커맨드는 기본적으로 내용 중 처음, 마지막 10줄만 출력함
    - 더 많거나 적은 줄을 출력하고 싶으면 옵션 n 사용
