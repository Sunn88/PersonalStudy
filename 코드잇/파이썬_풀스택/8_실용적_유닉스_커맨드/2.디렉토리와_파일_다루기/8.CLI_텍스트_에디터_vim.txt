파일 내용 작성
- GUI 환경 : Microsoft word, VS Code, 메모장 등 사용 가능
- CLI 환경 : Vim
    - 엄청나게 많은 기능과 단축키가 있음
    - 모든 걸 마우스 없이 키보드로 해결해야 함

Vim의 4가지 사용 모드
- 일반 모드(Normal Mode) : 커서 이동, 텍스트 붙여넣기, 작업 취소
    - i 입력시 입력 모드로
    - v, V 입력시 비주얼 모드로
    - : 입력시 명령 모드로
    - esc 입력시 다시 일반 모드로
- 입력 모드(Insert Mode) : 텍스트 입력
- 비주얼 모드(Visual Mode) : 텍스트 블록 지정, 텍스트 복사
- 명령 모드(Command Mode) : 내용 저장, Vim 종료
