컴퓨터 : Windows, macOS, Linux
핸드폰 : Android, iOS

Windows, Android : 사용자 설정을 쉽게 바꿀 수 있음
- 편리하지만 바이러스에 취약

macOS, iOS
- 보안이 철저한 대신 환경설정의 선택지가 제한되어 있음

용도에 따른 분류
- 데스크톱/랩톱 : Wondows >> macOS > Linux
- 모바일 : Andoid >> iOS > KaiOS
- 서버 : Ubuntu > Unix > CentOS
    - 안정성, 속도, 효율이 가장 중요
- 임베디드 : 실시간 운영 체제(Real-time operating system, RTOS)
    - 종류나 환경이 다양해서 운영 체제가 매우 많음
    - 무조건 빠른 것 보다 정해진 시간 이내에 안정적인 것이 더 중요
