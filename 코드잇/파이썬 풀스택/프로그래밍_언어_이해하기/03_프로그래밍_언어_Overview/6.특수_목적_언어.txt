특별한 목적을 위한 프로그래밍 언어(Domain-specific programming language)
ex) R, SQL, HTML, CSS 등의 언어

R : 주로 통계 소프트웨어 개발과 데이터 분석 및 시각화에 널리 사용됨
- 특히 통계나 빅데이터 분석에 필요한 데이터 전처리, 분석, 수학적인 계산과 그래프를 그리는 등의 기능에 특화됨

SQL : 데이터베이스를 관리하기 위한 목적의 프로그래밍 언어
- 데이터베이스 " 데이터를 모아둔 공간
- 프로그래머들은 SQL을 사용해서 데이터베이스를 설계하거나 운영하고 관리함
- 요즘에는 데이터가 워낙 많이 활용되기 때문에, 개발자 뿐 아니라 마케터나 데이터 분석가들이 SQL을 사용하여 서비스의 데이터(매출, 고객 정보 등)를 분석하곤 함

HTML, CSS : 웹 페이지를 만들 때 사용
- HTML : 주로 웹 페이지의 구조와 레이아웃을 담당
- CSS : 웹 페이지의 시각적인 표현을 담당
- 주로 여기에 JavaScript를 함께 사용해서, 웹 페이지가 사용자와 상호 작용할 수 있게 만들기도 함

그 외의 특수 목적 언어
- MATLAB: 공학 시뮬레이션용 프로그래밍 언어
- Csound: 소리와 음악 합성에 사용되는 프로그래밍 언어
- Scratch: 블록을 조립하여 쉽게 자기 자신의 이야기나 게임 그리고 애니메이션을 만들 수 있는 교육용 프로그래밍 언어
