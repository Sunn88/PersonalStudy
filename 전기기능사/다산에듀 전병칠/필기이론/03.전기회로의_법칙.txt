1. 옴의 법칙
한 회로에서 전류, 전압, 저항의 관계
V = I/R
I = R/V
R = V/I

저항의 계산
직렬 : R = R1 + R2
병렬 : R = 1/((1/R1) + (1/R2)) = R1R2/(R1 + R2)

전압의 분배
공급된 전압은 각 저항의 전압 강하의 합과 같다 -> 키르히호프 2법칙(전압)
V1 = IR1, V2 = IR2, V3 = IR3
V = V1 + V2 + V3
R = R1 + R2 + R3

전류의 분배
임의의 회로에서 한 점에 들어온 전류와 나간 전류의 대수합은 같다
I1 = V/R1, I2 = V/R2, I3 = V/R3
I = I1 + I2 + I3
R = 1/(1/R1 + 1/R2 + 1/R3)

I1 = I * R2/R1R2
I2 = I * R1/R1R2

중첩의 정리
전압원 -> 단락
전류원 -> 개방
