전지의 접속
기전력E, 저항r, R
I = V/R = E/(r + R)
I = V/R = (E1 + E2 + E3)/(r1 + r2 + r3 + R)
I = nE/(nr + R)

전류와 전압 및 저항의 측정
전압계 : 부하에 병렬로 연결
전류계 : 부하에 직렬로 연결
분류기 : 전류계의 측정 범위를 넓히기 위해 사용
배율기 : 전압계의 측정 범위를 넓히기 위해 사용
휘트스톤브리지 : 저항 4개를 마름모 모양으로 연결
- 브릿지 평형 : 대각선 저항간의 곱이 서로 같을 때 검류계에 전류가 흐르지 않음

* 내부저항과 외부저항이 같을 때 최대전력이 전달됨
