전기량 : 정전용량 * 전압
Q = C * V
정전용량 : C 커패시턴스 [F 패럿]
- 면적 A를 크게 하면 커짐
- 간격 l을 좁게 하면 커짐
- 유전율 엡실론을 크게 하면 커짐
C = A * 엡실론/l [F]

W = 1/2 * QV = 1/2 * CV^2 [J]

콘덴서
- 직렬연결
    - C0 = C1C2/(C1 + C2)
    - C0 = 1/(1/C1 + 1/C2 + 1/C3)
    - V1 = V * C2/(C1 + C2)
    - V2 = V * C1/(C1 + C2)
- 병렬연결
    - C0 = C1 + C2 
    - C0 = C1 + C2 + C3
    - Q1 = C1 * V
    - Q2 = C2 * V
